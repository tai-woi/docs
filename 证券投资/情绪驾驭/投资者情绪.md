周期因素
====

熊市底部
----

+ 悲观绝望期
+ 主导因素：*恐惧、绝望、风险厌恶*  
+ 关键驱动：  
  + 对极端负面事件（破产、违约、政策失误）的恐慌定价  
  + 长期下跌后的“创伤后应激反应”，忽视任何利好  
  + 流动性挤兑（强制平仓、基金赎回）主导情绪  
+ 典型指标表现：认沽/认购比率极高、波动率指数飙升、散户调查看多比例低于20%


初步复苏
----

+ 怀疑增长期
+ 主导因素：*怀疑、谨慎乐观、选择性相信*  
+ 关键驱动：  
  + 对基本面改善（盈利拐点、政策托底）的认知滞后  
  + “这次不同”心理：认为反弹不可持续，逢高抛售成主流  
  + 专业投资者先行，散户仍受熊市记忆压制  
+ 典型指标表现：融资余额缓慢回升、新增开户数仍低迷、但聪明钱流入明显


牛市中期
----

+ 乐观确认期
+ 主导因素：*趋势跟随、自我归因、合理化*  
+ 关键驱动：  
  + 盈利增长或流动性持续改善被市场广泛认可  
  + 媒体与分析师上调观点形成正反馈循环  
  + 情绪从“谨慎看好”转变为“回调就是买入机会”  
  + 出现第一批“故事驱动型”题材，估值容忍度提高  
+ 典型指标表现：换手率温和上升、隐含波动率稳定在低位、融资余额趋势性增长


牛市后期
----

+ 狂热与贪婪期
+ 主导因素：*过度自信、害怕踏空、社会比较*  
+ 关键驱动：  
  + 财富效应扩散（周围人赚钱的故事推动新资金入市）  
  + 估值逻辑被宏大叙事取代（“百年变革”“黄金十年”）  
  + 杠杆使用普遍化，散户开始向亲友推荐股票  
  + 对利空反应迟钝，对利好过度解读  
+ 典型指标表现：认沽/认购比率极低、新增开户数爆发、社交媒体“晒单”盛行、波动率指数异常平静


顶部区域
----

+ 分化与急躁期
+ 主导因素：*焦虑、急躁、非理性坚持*  
+ 关键驱动：  
  + 市场不再普涨，轮动加快，投资者担心“赶不上最后一程”  
  + 低质量股票也开始补涨（垃圾股、ST股活跃）  
  + 内部人（高管、大股东）开始减持，但散户情绪仍亢奋  
  + 出现“情绪背离”：调查看多比例极高，但价格开始滞涨  
+ 典型指标表现：涨跌家数比率恶化但指数未跌、期货从升水转贴水、融资余额增速放缓


下跌初期
----

+ 否认与希望期
+ 主导因素：*否认、抱有侥幸、逢低抄底*  
+ 关键驱动：  
  + 首次大跌被解释为“技术性调整”  
  + 投资者急切寻找支撑理由（基本面并未变差）  
  + 越跌越买现象出现，试图摊平成本  
  + 媒体仍以“回调即买点”为主调  
+ 典型指标表现：看多调查比例仍偏高、认沽比率未明显上升、散户资金净流出不明显


熊市中期
----

+ 恐慌与投降期
+ 主导因素：*恐慌、后悔、风险规避*  
+ 关键驱动：  
  + 前期“抄底”资金被套，情绪由希望转为愤怒  
  + 流动性突然枯竭（卖不掉导致的情绪崩溃）  
  + 强制卖出（止损、赎回、追保）成为主要驱动  
  + 任何消息都被解读为利空，利好被忽视  
+ 典型指标表现：波动率指数飙升、融资余额断崖式下跌、认沽比率快速上升

熊市后期
----

+ 绝望与麻木期
+ 主导因素：*绝望、放弃、无情绪*  
+ 关键驱动：  
  + 成交量极度萎缩，交易意愿降至冰点  
  + 投资者不再谈论股市，研报覆盖减少  
  + 对政策或利好完全无反应（“叫好不叫座”）  
  + 情绪从“恐惧”转为“厌倦”，市场对任何信息免疫  
+ 典型指标表现：换手率极低、新增开户数冰点、散户调查看多比例长期在低位徘徊（但不再下降）


核心变化规律
----

+ 恐惧驱动底部（下跌中继 → 绝望底部）  
+ 怀疑驱动早期复苏（底 → 初升段）  
+ 趋势驱动中期牛市（主升浪）  
+ 贪婪与害怕错过驱动后期牛市（末升段）  
+ 否认驱动顶部初跌（转折点）  
+ 恐慌驱动主跌段  
+ 麻木驱动最后底部

理解这一情绪演变链条，有助于识别当前市场大致处于周期的哪个位置，避免在贪婪期追高、在绝望期割肉。



考量指标
====

成交量与成交额
----

+ 成交量异常放大：观察个股或指数成交量是否显著高于20日均量（如高出50%以上），常伴随行情阶段性高点。
+ 成交额占流通市值比例：全市场成交额/总流通市值，比例过高（如A股超过5%）说明交易过度活跃。
+ 换手率：
  + 日换手率>10%：短线过热
  + 周换手率>30%：筹码交换过快，需警惕
+ 量价背离：股价创新高但成交量未能同步放大，或缩量上涨，提示热度可能不可持续。


资金流向与杠杆资金
----

+ 北向资金（陆股通）：
  + 单日净买入超100亿元（A股标准）为情绪高涨
  + 连续5日以上净买入需关注过热风险
+ 融资余额变化：
  + 融资余额单周增幅>5%说明杠杆资金涌入
  + 融资买入额占A股成交额比例>10%为过热信号
+ 主力资金净流入/流出：连续多日大额净流入后出现单日大幅净流出，常是热度拐点。


情绪与调查指标
----

+ 恐慌与贪婪指数（如CNN Money Fear & Greed Index）：
  + 数值 > 75：贪婪 → 过热
  + 数值 < 25：恐惧 → 过冷
+ 投资者情绪调查（如AAII美国个人投资者协会调查）：
  + 看多比例 > 55% 且看空比例 < 20% → 过热
+ 新高新低指数（NH-NL）：
  + 创20日新高股票数减去创20日新低股票数，差值显著为正且不断扩大说明热度上升。


技术指标
----

+ RSI（相对强弱指数）：
  + 日线RSI > 80：超买（过热）
  + 周线RSI > 85：严重过热
+ 布林带：价格连续多日紧贴或突破上轨运行，且开口扩大。
+ MACD：DIF与DEA在零轴上方高位再次金叉，或出现顶部背离。
+ KDJ：J值>100并钝化，或K值>90连续5日以上。


衍生品市场
----

+ 期权认沽/认购比（Put/Call Ratio, PCR）：
  + PCR（成交量） < 0.7 → 过度乐观
  + PCR（持仓量） < 0.5 → 严重过热
+ VIX指数（波动率指数）：
  + VIX处于历史低位（如 < 12） → 自满情绪，可能反向信号
  + VIX急速拉升则反映恐慌，而非热度
+ 期货升贴水：
  + 股指期货持续大幅升水（如年化升水>10%） → 情绪亢奋


市场宽度与广度
----

+ 上涨下跌家数比：
  + 全市场上涨家数/下跌家数 > 3 且持续多日 → 过热
  + 单日比值 > 8 为极度亢奋
+ 腾落线（ADL）：指数创新高但ADL未同步新高，出现背离则热度虚高。
+ 创60日新高股票占比：占比 > 30% 为过热区域。


行为与新闻热度
----

+ 百度指数 / 微信搜索指数：关键词（如“股票开户”“牛市”）搜索量较均值上升2倍以上。
+ 社交媒体情绪：通过NLP分析微博、股吧、Reddit的看多帖子比例。
+ 新开户数：周新增投资者环比增长超50%为热度高峰信号。
+ 基金发行：单只偏股基金募集规模超百亿且比例配售频现 → 情绪顶点。


综合分析流程
----

1. 多维度交叉验证：不依赖单一指标，至少选3个维度同时发出过热信号再确认。
2. 时间周期对比：将当前数值与自身历史（近1年、3年）百分位对比，>80%为过热。
3. 区分整体与局部热度：用市场宽度区分是普涨（整体热）还是龙头抱团（局部热）。
4. 领先滞后判断：情绪指标往往领先价格拐点1-3天，资金流向指标滞后半天至1天。
5. 反向思考：极端一致乐观（如几乎无看空观点）往往是卖出信号。


注意事项
----

+ 热度信号在强趋势市中可能持续“钝化”（如牛市RSI长期>80）。
+ 政策干预市场（如涨跌停板、T+0限制）会扭曲部分指标有效性。
+ 不同市场（A股、美股、港股）的阈值需分别设定。
