六大类型
====

趋势型指标
----

+ 核心逻辑：跟随价格趋势，过滤短期噪音，识别市场主要运动方向。
+ 代表性指标：移动平均线（MA）、指数平滑异同移动平均线（MACD）、平均趋向指数（ADX）
+ 优点
  + 在强趋势行情中能捕捉主升/主跌浪，盈利概率高。
  + 提供明确的买卖信号（如金叉、死叉、突破）。
  + 帮助投资者“顺势而为”，避免逆势交易。
+ 缺陷
  + 在震荡（盘整）市场中频繁产生虚假信号，导致连续亏损。
  + 具有滞后性——价格明显转折后指标才会跟随变化。
  + 参数设置敏感：短期参数易被洗盘，长期参数反应过慢。


震荡型指标
----

+ 核心逻辑：摆动指标衡量价格相对于一定时期内波动范围的位置，识别超买超卖和背离。
+ 代表性指标：相对强弱指数（RSI）、随机指标（KDJ）、顺势指标（CCI）
+ 优点
  + 在震荡市中表现优异，能较好捕捉高低点。
  + 信号出现较快，比趋势指标提前预警反转。
  + 背离（价格与指标反向）往往预示趋势减弱或反转。
+ 缺陷
  + 在单边强势趋势中会长期停留在超买/超卖区，导致过早反向交易而踏空或亏损。
  + 钝化现象：极端行情下指标失去弹性，无法提供有效信号。
  + 背离信号有时会多次出现才有效，容易误判。


成交量型指标
----

+ 核心逻辑：通过成交量的变化验证价格行为的可靠性，反映资金参与意愿。
+ 代表性指标：能量潮（OBV）、成交量加权平均价（VWAP）、资金流量指数（MFI）
+ 优点
  + 价格信号配合成交量确认时，可信度大幅提高。
  + 可以提前发现资金流入/流出迹象，例如价平量增、量价背离。
  + 有助于识别趋势的强弱和健康程度。
+ 缺陷
  + 在成交不活跃的个股或小盘股中容易失真（存在对倒、操纵）。
  + 单独使用成功率较低，通常需要配合价格指标。
  + 极端缩量或爆量后，信号解释存在多重可能性。


能量型指标
----

+ 核心逻辑：动量指标测量价格上涨或下跌的速度与力度，判断趋势的衰竭或加速。
+ 代表性指标：动能指标（Momentum）、变动率指标（ROC）、心理线（PSY）
+ 优点
  + 领先性较强，能较早提示趋势可能变化。
  + 识别动量衰竭有效，帮助规避追高风险。
  + 适合捕捉加速段和转折点。
+ 缺陷
  + 容易受价格短期脉冲影响，出现虚假加速/衰竭信号。
  + 在横盘市场中频繁无意义波动，干扰判断。
  + 对参数敏感，不同周期下信号差异极大。


波动型指标
----

+ 核心逻辑：波动率指标衡量价格波动的幅度与速度，判断市场情绪和未来可能的价格区间。
+ 代表性指标：布林带（Bollinger Bands）、平均真实波幅（ATR）、肯特纳通道（KC）
+ 优点
  + 识别市场即将突破或变盘（波幅压缩到极致时）。
  + 帮助设置动态止损和止盈（例如基于ATR的止损）。
  + 反映市场恐慌或平静程度（如布林带收口）。
+ 缺陷
  + 不指示价格方向，只指示波动范围。
  + 波动率突然扩张往往是价格已大幅变动之后，信号滞后。
  + 单独使用无法产生买卖点，需要与其他指标结合。


支撑压力型指标
----

+ 核心逻辑：周期/带状指标通过统计历史价格或数学计算，画出潜在的支撑、阻力或成本区域。
+ 代表性指标：斐波那契回调线、轴心点（Pivot Points）、趋势线（由K线衍生）
+ 优点
  + 提前给出关键价格水平，便于挂单交易。
  + 位置相对稳定，不会随着短期价格跳动频繁变化。
  + 在价格接近这些区域时，决策参考价值高。
+ 缺陷
  + 在强趋势行情中，支撑/压力常常被轻易击穿。
  + 不同方法（如均线、斐波那契、枢轴点）给出的点位不同，易产生选择困惑。
  + 过去形成的有效位置在未来可能完全无效。



A股偏好
====

A股市场具有一些鲜明特征：*政策影响大、散户占比高、情绪波动剧烈、存在明显的“风口/板块轮动”以及涨跌停板制度*。因此，单纯照搬欧美成熟市场的指标组合，效果往往不佳。

核心环境偏好
----

+ *偏好环境*：震荡上升趋势或结构性行情（指数涨幅不大，但板块/个股活跃）
  + 原因：A股全面大牛市短暂，深度熊市杀伤力强；多数时间呈现板块轮动的震荡上行或区间波动。
+ *应高度警惕的环境*：情绪极度一致后的缩量下跌（例如连续涨停后的放量跌停开板）
  + 原因：涨跌停板会导致情绪超调，技术信号容易失效或极端化。
+ *不宜使用单纯技术指标的时期*：重大政策公布前、年报/季报密集披露期、长假前
  + 原因：基本面或资金避险需求主导，技术面容易失真。


特殊指标参数调整
----

+ 均线周期
  + 短期：5、10、20日；中期：60日；长期：120日（半年线）、250日（年线）
  + 理由：A股趋势延续性不如美股，零轴上方死叉需警惕
+ MACD
  + 参数（12,26,9）不变，但更*重视柱体面积与零轴关系*
  + 理由：A股趋势延续性不如美股，零轴上方死叉需警惕
+ RSI / KDJ
  + 超买/超卖阈值收窄：RSI超买≥70→65，超卖≤30→35；KDJ默认（9,3,3）
  + 理由：因A股波动更大且易钝化，稍保守信号更可靠
+ 成交量指标
  + 量比 >1.5 且持续；换手率 3%-10% 为活跃区间
  + 理由：识别真实资金推动，避免对倒陷阱
+ 波动指标 (ATR)
  + 周期常用14日，但需要剔除涨跌停板造成的异常波幅
  + 理由：涨停日ATR暴增会扭曲止损距离


分场景指标组合示例
----

### 上升趋势或反弹行情

+ 适用时机：大盘站稳60日均线，且某一板块（如AI、新能源）持续出现涨停股。
+ 主导指标：均线系统（5/20/60） + 成交量（换手率/量比）
+ 辅助指标：MACD（确认趋势强度）
+ 回避指标：KDJ超买信号（强势股KDJ可长时间高于80）
+ 组合动作：
  + 选择股价在20日均线之上，60日均线走平或上行的个股。
  + 单日换手率 > 5% 且量比 > 1.2，视为有资金参与。
  + 当股价回踩20日均线不破，且MACD柱状线由绿翻红或维持红色，是常见买点。
  + 跌破20日均线且连续3日无法收回，则离场。

### 连续下跌后的震荡修复

+ 适用时机：指数或个股连续下跌后，出现长下影线、缩量十字星，且远离5日均线（乖离率较大）。
+ 主导指标：RSI（参数7或9） + 成交量（地量）
+ 辅助指标：布林带（观察是否触及下轨）
+ 关键确认：日线RSI(7) 低于20（而非30），且第二日回升至25以上；同时成交量萎缩至前期均量的60%以下。
+ 组合动作：
  + 当RSI(7)从20以下回升，且当日收盘价高于前日最低价，可轻仓试探。
  + 反弹目标位：一般先看5日均线和10日均线的压力区。若反弹至20日均线附近且成交量跟不上，则平仓。
  + 注意：A股超跌反弹失败概率高，建议快进快出，止损设在前低下方2%-3%。

### 识别涨停板后的真伪强弱

+ 适用时机：个股出现首板或二连板后，需要判断是否值得追或持有。
+ 主导指标：封单量/成交额（盘口） + 换手率
+ 辅助指标：布林带（观察是否突破上轨开口）
+ 组合动作：
  + 良性信号：涨停板封单 > 当日成交额（例如封单5亿，成交额3亿），换手率控制在5%-15%之间（非过高），且涨停时间早于上午10:30。
  + 警告信号：尾盘勉强封板、封单极小、换手率超过25%（出货嫌疑）。
  + 次日观察：高开3%-7%且半小时内不补缺口，可持有；低开或平开则迅速减仓。

### 规避“炸板”与“核按钮”（短线风险控制）

+ 适用时机：手中持有涨停股，或计划打板。
+ 主导指标：分时图量能 + 涨跌停家数（市场情绪指标）
+ 辅助指标：15分钟KDJ（超短线敏感）
+ 组合动作：
  + 如果当日涨停家数显著少于昨日（例如从100家骤降至30家），减少打板操作。
  + 持仓股若在9:35-9:50期间未能继续封板，且分时成交量迅速萎缩，可考虑部分止盈。
  + 若出现“天地板”或盘中跌幅超过7%且封死跌停，不必补仓，等待次日集合竞价判断。


两种主要模式
----

### 趋势跟踪

+ 核心指标组合：20/60均线 + 成交量 + MACD
+ 时间框架：日线为主
+ 止损纪律：跌破20日均线
+ 持仓周期：数周至数月

### 超短打板

+ 核心指标组合：换手率 + 封单量 + 5日均线 + 分时图
+ 时间框架：分时/30分钟
+ 止损纪律：收盘破5日均线
+ 持仓周期：1-5天


两个常见陷阱
----

### 利好出尽是利空

当指标走好（如MACD金叉、放量突破）时，如果该股正好在公告重大利好后的第三天，反而可能是主力借机出货。此时应观察*成交量是否持续*，若次日缩量30%以上，则警惕。

### 流动性分化

小市值A股（例如日成交额低于5000万）的技术指标极易被操纵，例如对敲放量、假突破。对于这类个股，*建议降低指标权重，提高风险控制*（如严格止损或不做）。反之，对于日成交额10亿以上的龙头股，技术指标可信度高。



板块因素
====

不同板块因属性（市值、流动性、波动率、基本面驱动因素）不同，技术指标的应用侧重点和参数需要有所调整。

大盘蓝筹板块
----

+ 金融、消费、能源龙头股
+ 特点：市值大、波动相对平缓、流动性好、走势与宏观基本面相关性强。
+ 常用指标及用法：
  + 移动平均线：中长周期为主（如60日、120日、250日）。金叉/死叉信号可靠性高，用作趋势确认。
  + MACD：零轴上方/下方运行需持续关注；柱状线背离用于捕捉中期顶底。
  + RSI：超买超卖区域可适当放宽（如80/20），因趋势性强，过早反向操作风险高。
  + 成交量：放量突破关键均线或平台是重要买入信号；缩量调整视为健康。
  + 布林带：价格常沿中轨（20日均线）运行，触及上下轨后回归概率中等，需结合趋势判断。


中小创板块
----

+ 中小板、创业板、科创板
+ 特点：高波动、高换手、资金情绪驱动强、易出现急涨急跌。
+ 常用指标及用法：
  + RSI：设置较短周期（如7日或9日），超买超卖阈值可设70/30，用于捕捉短线转折。
  + KDJ：高频使用，J值上穿/下穿100/0区域需警惕反转；金叉死叉信号在震荡市更有效。
  + 成交量（VOL）：量比（当前成交量/5日均量）大于1.5-2倍视为活跃；放量滞涨需警惕出货。
  + BOLL布林带：价格频繁触及上下轨，开口扩张常预示趋势加速，缩口后易变盘。
  + 均线：短期均线组合（5、10、20日）为主，跌破10日线视为短线转弱信号。


科技成长板块
----

+ 半导体、AI、新能源、生物医药等
+ 特点：高估值敏感性、事件驱动（技术突破、政策、订单）、波动剧烈且易形成趋势。
+ 常用指标及用法：
  + 均线多头/空头排列：判断主升浪或主跌浪；20日均线作为生命线，跌破需止盈/止损。
  + MACD：重点利用柱状线（红绿柱）的加速与减速，而非仅看金叉死叉。柱子缩短但价格新高/新低可能为背离信号。
  + SAR抛物线指标：行情连续时，SAR翻红/翻绿可作为右侧交易止盈止损点。
  + ATR平均真实波幅：用于动态设置止损位（如3倍ATR），适应板块高波动特性。
  + RSI与MACD联合背离：顶背离/底背离信号在板块主升/主跌末期可靠性较高。


周期性板块
----

+ 有色、煤炭、钢铁、化工、航运
+ 特点：与大宗商品价格、宏观库存周期、供需数据强相关，波段明显。
+ 常用指标及用法：
  + 成交量 + 价格行为：放量突破长期盘整区间是重要启动信号；缩量回踩前期高点/支撑可介入。
  + CCI顺势指标：CCI从-100下方回升至+100以上，代表强势启动；从+200以上回落需警惕阶段顶部。
  + 均线乖离率（BIAS）：当价格大幅偏离长期均线（如120日）时易回归，适合做波段高抛低吸。
  + OBV能量潮：股价震荡但OBV创新高，预示后续上涨动力；背离信号比价格本身更早。
  + MACD：零轴下方金叉后能否快速拉上零轴，是强弱分界。


金融板块
----

+ 银行、保险、券商、多元金融
+ 特点：低波动（券商除外）、高股息、受利率和监管政策影响大。
+ 常用指标及用法：
  + 均线：长周期（60日、120日）作为主要支撑/压力；券商板块可缩短至20日。
  + RSI：不易触发极端值，长时间低于30可能意味着价值区域，高于70但趋势未坏可持有。
  + MACD：零轴附近金叉/死叉信号价值高，远离零轴后易钝化。
  + 成交量：银行/保险关注“地量见地价”信号；券商需关注“倍量启动”，是其行情先行指标。
  + ADX平均趋向指数：ADX低于20时无趋势，高于25且上升表示趋势形成（尤其适用券商）。


防御性板块
----

+ 公用事业、医药、必需消费
+ 特点：抗跌性强、熊市中相对强势、波动小、资金抱团属性明显。
+ 常用指标及用法：
  + 均线：熊市中看60日线支撑；牛市中看20日线。跌破重要均线往往是抱团松动信号。
  + 布林带：价格长期沿中轨与上轨之间运行（强势）；跌破中轨需警惕转弱。
  + RSI：在40-70之间反复震荡，极少进入超买区；突破70可视为强势延续。
  + 成交量：上涨无需连续放量，缩量上涨属于常态，但放量下跌是危险信号。
  + DMI：+DI（多方方向线）在-DI上方且ADX上升，代表防御资金主导趋势。


次新股板块
----

+ 特点：流通盘小、无历史套牢盘、筹码结构简单、技术指标易失真。
+ 常用指标及用法：
  + 换手率：最重要指标。开板后换手率从高位回落至15%-20%以下且价格企稳，是博弈反弹的前提。
  + 成交量：极度萎缩（相对上市首日或开板日）后再次温和放大，可能启动。
  + 均线：5日、10日、20日线为主，但历史参考意义弱，需结合次新股整体情绪周期。
  + RSI / KDJ：易钝化，在超买区持续钝化反而可能更强，不建议作为主要反转依据。
  + 形态分析：W底、旗形整理、开板后缩量平台等经典形态比指标更可靠。


ETF与行业指数板块
----

+ 特点：分散个股风险、趋势性更强、受成分股整体走势影响。
+ 常用指标及用法：
  + 均线系统：有效性高于个股。60日线作为中期趋势分水岭，120日线为牛熊线。
  + MACD：红绿柱面积变化判断趋势强度，零轴上方回踩不破零轴视为强势整理。
  + 成交量：ETF份额变化（场内流通份额）比日成交量更有参考价值。
  + BOLL布林带：行业指数在布林带中轨上方运行，做多；跌破中轨至下轨，回避。
  + RSI：行业ETF突破70后若持续钝化，并非危险，可能主升浪；回调不破50为强势。


通用注意事项
----

+ 指标数量：单只个股或板块，建议使用2-3个互补指标（如趋势+摆动+量能），而非堆砌。
+ 时间周期：日线用于主判断，周线确认大趋势，60分钟线用于进出场时机。
+ 市场环境：震荡市中KDJ、RSI等摆动指标更有效；趋势市中MACD、均线系统更有效。
+ 板块轮动：技术指标失效时，往往是板块风格切换的前兆，可结合资金流向（北向、龙虎榜）辅助判断。
+ 业绩与事件：财报、行业政策、产品价格变动会短期内完全压制技术信号，需提前规避或结合基本面。

不同板块使用技术指标的核心原则是：*高波动板块侧重摆动与量能，低波动板块侧重趋势与通道；流动性好的板块均线有效，流动性差的板块慎用指标背离。*
