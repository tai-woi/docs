新手炒股，最大的坑不是亏钱，而是“用错误的方式亏掉不该亏的钱”。股市是个“认知变现”的地方，下面这10个坑，是我见过最多人用真金白银填出来的，希望能帮你绕过去。

用生活钱炒股
====

这是最致命的一条。拿生活费、房贷钱、结婚钱、甚至借来的钱炒股，一旦市场波动，你的心态会瞬间崩溃。因为输不起，所以会做出所有错误决策：跌一点就慌，涨一点就跑，完全无法理性思考。

避坑法：拿3-5年用不上的闲钱投资。这笔钱没了，不影响你的生活品质。


把运气当本事
====

刚入市时，正好赶上行情好，随便买什么都涨，于是觉得自己是“股神”。这是市场给你设下的最甜美的陷阱。因为接下来，你会把侥幸赚来的钱，凭实力亏回去。

避坑法：建立自己的交易记录，每一笔买卖都写清楚“为什么买、为什么卖”。定期复盘，把靠运气赚的和靠逻辑赚的分开。


亏了扛，赚点跑
====

这是新手最典型的亏损模型（“倒金字塔”亏损）：股票涨了10%赶紧卖，生怕利润回吐；股票跌了20%却死扛着不卖，还不断补仓想摊平成本。结果小赚大亏，一只股票套牢30%-50%。

避坑法：止损和止盈要对称。买入前就想好：跌多少我认赔（比如-8%），涨到哪我考虑卖。亏损头寸绝不加仓，加仓只能加在盈利的头寸上。


爱上你的股票
====

因为买了某只股票，就开始只看它的利好消息，对利空视而不见，甚至到处找理由证明自己是对的。股票不会因为你买了它就涨，它也不认识你。

避坑法：把股票当成代码和数字，它是你的交易工具，不是你的恋人。当基本面或趋势走坏时，果断分手（止损）。


信息依赖和大师崇拜
====

天天刷股吧、看短视频，听“老师”荐股，或者迷信某个技术指标。要知道，当你在公开渠道看到某个“确定性的机会”时，这通常就是主力想让你看到的。

避坑法：建立自己的信息过滤机制。只看一手信息（公司公告、财报），不看二手解读（小道消息、付费荐股）。任何给你“必涨牛股”的人，非骗即割。


频繁交易
====

把炒股当上班，总觉得每天不操作就手痒，佣金和印花税交了一大堆。A股是T+1且交易成本不低，频繁交易不仅累，而且胜率会随着次数增加而趋近于50%，扣除手续费后必亏。

避坑法：把交易频率降下来。用周线或月线看趋势，而不是分钟线。如果你没想好持有3个月，就连3分钟都不要持有。


永远满仓
====

看不得账户里有现金，总觉得现金会贬值，必须换成股票才安心。结果是，当真正的暴跌来临时，你没有子弹抄底；当机会来临时，你被套牢动弹不得。

避坑法：仓位管理比选股更重要。学会在行情不明朗时空仓或轻仓等待。空仓不是踏空，是主动选择的安全。


杂货铺
====

资金不多，却买了十几二十只股票，美其名曰“分散风险”。结果行情好时涨得慢，行情差时全盘皆绿，根本看不过来。

避坑法：新手3只以内足够。集中仓位到你最有把握的标的上，深度研究一只，好过浅尝辄止十只。


只买便宜股
====

专门买股价低、跌了很多的股票，觉得“便宜”安全，涨起来空间大。但低价≠低估值，很多低价股是因为公司基本面差，跌了还能再跌，甚至退市。

避坑法：看市值和估值（PE/PB），不看绝对价格。好公司贵有贵的道理，垃圾公司再便宜也是负担。


策略与行情错位
====

牛市用追涨策略，熊市用抄底策略，震荡市用网格策略。新手往往一招鲜，在震荡市里追涨杀跌，两面挨耳光。

避坑法：先判断市场环境（是上涨、下跌还是震荡），再选择对应策略，做好仓位管理。如果看不清，就空仓观望。

另一种方案，*策略固化法*：放弃择时，只做一种策略，__或根据预先设定的仓位比例分配策略__。

+ 趋势交易：永远只做追涨。牛市大赚，震荡市小亏，熊市空仓。靠牛市的大赚覆盖震荡市的小亏。
+ 价值投资：永远只做抄底。熊市大买，牛市卖出。靠熊市的廉价筹码获取超额收益。
+ 网格交易：永远只做震荡。靠高抛低吸，最怕单边大牛或大熊，所以必须设好上下边界。


最后的建议
====

最后送你一句很实在的话：新手的第一目标不是赚钱，而是“活下来”。用最小的成本（轻仓）去积累经验，把前两三年当作“交学费期”，只要不伤本金，你就跑赢了绝大多数人。
