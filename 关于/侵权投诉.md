总则
====

本网站高度重视知识产权保护及个人/企业合法权益维护。为规范侵权投诉程序，根据《中华人民共和国民法典》《中华人民共和国著作权法》《中华人民共和国商标法》《中华人民共和国反不正当竞争法》《中华人民共和国个人信息保护法》《网络信息内容生态治理规定》等法律法规，特制定本侵权投诉指引。

任何单位或个人认为本网站所发布的内容（包括但不限于文字、图片、音频、视频、链接等信息）侵犯其合法权益的，均可按照本指引提交投诉材料。本指引同时适用于针对用户通过评论、论坛、群组等功能板块发布内容的投诉。

特别提示： 如您因被侵权而遭受重大损失或面临紧迫风险，请在提交投诉的同时，及时通过诉讼、财产保全等法律途径寻求救济，本网站将依法配合司法机关的调查和执行工作。


侵权投诉范畴
====

适用的侵权类型
----

以下类型的侵权投诉适用本指引处理：

1. 侵犯著作权：未经著作权人许可，复制、发行、表演、放映、广播、汇编、通过信息网络向公众传播其作品等行为。
2. 侵犯商标权：未经商标注册人许可，在同一种商品或类似商品上使用与其注册商标相同或近似的商标，容易导致混淆的行为。
3. 侵犯专利权：未经专利权人许可，实施其专利的行为。
4. 侵犯企业名称权/商号权：擅自使用他人有一定影响的企业名称（包括简称、字号等）。
5. 侵犯名誉权：以侮辱、诽谤等方式损害公民或法人的名誉。
6. 侵犯隐私权：泄露、公开他人私密信息（如身份证号、住址、电话号码、行踪轨迹等）或侵扰他人私人生活安宁。
7. 侵犯肖像权：未经同意，制作、使用、公开他人肖像。
8. 泄露商业秘密：未经权利人许可，披露、使用或允许他人使用其技术信息、经营信息等商业秘密。
9. 其他合法权益：法律法规规定的其他侵犯民事权益的行为。


不适用的内容
----

以下内容不属于本指引项下的侵权投诉范畴，请您通过其他渠道处理：

1. 一般性违法或不良信息：内容涉嫌违反法律法规、社会主义制度、国家利益、公民合法权益、社会公共秩序、道德风尚、信息真实性等“七条底线”要求的（如涉黄、涉赌、涉毒、暴力恐怖、诈骗信息等），请通过本网站[论坛举报](https://github.com/tai-woi/docs/discussions/categories/%E4%B8%BE%E6%8A%A5)渠道提交。
2. 单纯意见分歧或主观不满：内容仅涉及对产品、服务、企业的一般性批评、吐槽或主观评价，不构成侮辱、诽谤或失实陈述的。
3. 竞争纠纷：涉及搜索引擎收录、排名算法、流量统计等平台运营机制的问题。
4. 合同纠纷：涉及与本网站之间的服务合同履行问题。
5. 行政或刑事举报：涉及违法犯罪行为的举报，请直接向公安机关、网信部门、市场监管部门等有权机关投诉举报。
6. 其他不属于民事侵权范畴的争议 。

投诉材料要求
====

基础材料
----

| 投诉人类型 | 所需材料 |
| :--- | :--- |
| 自然人 | 本人身份证或护照复印件（或电子扫描件），并签字确认 |
| 法人或其他组织 | 营业执照复印件（或电子扫描件），并加盖公章 |


权利证明材料
----

根据投诉类型，请提供以下相应权利证明：

+ 著作权：著作权登记证书、创作底稿、首次发表证明、原始拍摄/录制文件、授权协议等。
+ 商标权：商标注册证书。
+ 专利权：专利证书。
+ 名誉权/隐私权/肖像权：身份证明、相关权利归属证明。
+ 商业秘密：保密协议、技术/经营信息载体、采取保密措施的证明等。
+ 其他：能够证明权利归属的其他材料。


投诉内容材料
----

1. 被投诉内容的准确定位：须提供完整的网址链接（URL）及页面截图。
2. 侵权事实说明：须明确指出被投诉内容侵犯了您的何种具体权利，并说明理由。
3. 初步证据：能够初步证明侵权事实存在的材料。


保证声明
----

投诉人须在投诉材料中附以下声明，并亲笔签名或加盖公章：

> “本人/本单位声明：投诉内容真实、合法、有效，所提供材料均为真实可靠。本人/本单位保证，若因投诉内容不实而给本网站或第三方造成损失，本人/本单位自愿承担全部法律责任。”


材料形式要求
----

1. 所有材料须为*A4纸规格*，字迹清晰可辨认。
2. 涉外的投诉材料，建议经所在国公证机关公证并办理中华人民共和国驻该国使领馆认证。
3. 非中文材料，建议随附中文翻译件。


投诉处理流程
====

本网站处理侵权投诉遵循“接收—审核—处理—反馈”四个阶段，具体流程如下：

### 第一步：投诉提交

投诉人按照本指引第三条的规定，通过在线表单、邮件或信函方式提交完整的投诉材料。

### 第二步：材料审查

本网站在收到投诉材料后的~~5个工作日内~~对材料进行形式审查：

+ 材料完整的：正式受理，进入下一阶段；
+ 材料不完整的：通知投诉人在~~5个工作日内~~补充材料，逾期未补正或补正后仍不符合要求的，视为撤回投诉；
+ 明显不属于侵权投诉范畴的：驳回投诉，并告知理由或指引投诉人通过其他渠道反映。

### 第三步：受理与初步审核

正式受理后，本网站将对投诉内容进行实质性审核（期限为受理之日起~~15个工作日~~，情况复杂的可延长至~~30个工作日~~）：

+ 侵权事实明显：直接采取删除、屏蔽、断开链接等必要措施；
+ 侵权事实存疑或需要进一步核实：将投诉材料转送被投诉人，要求其在~~5-10个工作日~~内提交书面抗辩说明。

### 第四步：被投诉人抗辩

+ 被投诉人未在规定期限内提交抗辩说明：本网站将采取必要措施；
+ 被投诉人提交了有效抗辩说明：本网站将抗辩说明转送投诉人，并告知投诉人可依法向有关行政机关投诉或向人民法院起诉。

### 第五步：司法衔接

投诉人在收到转送的反通知后~~15日内~~未向有权机关投诉或起诉的，本网站可以终止已采取的必要措施，恢复被投诉内容。

### 第六步：结果反馈

处理完成后，本网站将以邮件或站内消息等方式同时通知投诉人和被投诉人，告知处理结果及依据。


重要提示
====

1. *材料真实性*：请务必确保投诉材料的真实性。若经核实存在恶意捏造事实、伪造证据等行为，本网站有权将相关材料移交公安机关处理，并依法追究投诉人的法律责任。 因错误投诉或虚假投诉给他人造成损失的，投诉人须依法承担侵权赔偿责任。
2. *及时处理*：对于有效投诉，本网站将在合理期限内尽快处理。但在材料不完整或事实不清的情况下，处理时间可能延长。
3. *非侵权内容处理*：对不属于侵权投诉范畴的举报内容，本网站将按照用户协议及相关管理规定处理，但不适用本指引规定的处理流程和时限。
4. *司法优先*：若本网站已对投诉内容采取必要措施，投诉人就同一侵权争议向人民法院起诉的，本网站将配合司法程序，但本网站对内容的处理仍将以生效法律文书为最终依据。
5. *联系方式变更*：投诉人联系方式发生变更的，应及时告知本网站，因无法联系导致无法处理的，后果由投诉人自行承担。
6. *保密义务*：本网站在处理投诉过程中获得的个人信息和商业秘密，将严格保密，仅用于投诉处理之必要目的。
7. *滥用投诉的后果*：对于反复提交虚假投诉、滥用投诉机制的用户，本网站有权采取限制其投诉权限乃至封禁账号等措施。


反通知机制
====

被投诉人对本网站采取的必要措施有异议的，有权通过上述联系方式提交反通知。反通知应包含以下内容：

1. 被投诉人的姓名/名称及联系方式；
2. 要求恢复的内容名称和网络地址；
3. 不构成侵权的初步证明材料；
4. 声明保证材料真实性的保证书。

本网站在收到有效的反通知后，会将反通知转送给投诉人，并告知其可以向有权机关投诉或向人民法院起诉。投诉人在收到转送的反通知后~~15日内~~未向有权机关投诉或起诉的，本网站可以终止已采取的必要措施。
