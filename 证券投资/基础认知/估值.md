主要估值方法
====

绝对估值法
----

+ *DCF（现金流贴现）*：预测未来自由现金流，用WACC贴现。最严谨，但参数敏感。
  + 适用行业：现金流稳定可预测的公用事业、消费龙头
  + 不适用行业：高不确定性、早期或周期性行业
  + 限制原因：假设敏感度高，未来现金流难准确预测
+ *DDM（股利贴现模型）*：以未来股利为贴现对象。适用于分红稳定公司。
+ *剩余收益模型（EVA）*：基于经济利润（超过资本成本的收益）。适合有隐蔽资产或非经常性损益的公司。
+ *调整现值法（APV）*：分开计算无杠杆项目价值与税盾价值。适合高负债或资本结构变动公司。


相对估值法
----

+ *P/E（市盈率）*：股价 ÷ 每股收益。最常用，但盈利为负时失效。
  + 适用行业：稳定盈利的消费、公用事业、部分科技
  + 不适用行业：周期行业、亏损科技、银行、早期生物科技
  + 限制原因：利润为负或剧烈波动
+ *P/B（市净率）*：股价 ÷ 每股净资产。适用于重资产或清算价值型公司。
  + 适用行业：金融、周期低谷、重资产工业
  + 不适用行业：轻资产科技、服务、品牌消费品
  + 限制原因：账面价值不反映无形资产（品牌、技术）
+ *P/S（市销率）*：股价 ÷ 每股销售额。适用于成长初期或亏损公司。
  + 适用行业：早期科技、零售、电商
  + 不适用行业：低利润率行业（如大宗商品贸易）
  + 限制原因：忽略成本结构和利润率差异
+ *EV/EBITDA（企业价值倍数）*：企业价值 ÷ 息税折旧摊销前利润。排除了资本结构、折旧、税率差异。
  + 适用行业：资本密集型行业（电信、能源、重工业）、有大量折旧摊销的行业
  + 不适用行业：高增长轻资产科技、金融
  + 限制原因：调整了资本结构差异，但忽略增长和营运资本需求
+ *PEG（市盈率相对盈利增长比）*：(P/E) ÷ 盈利增长率。用于高成长公司，弥补P/E未考虑成长性。
  + 适用行业：稳定增长且盈利为正的成长行业（部分消费、科技）
  + 不适用行业：周期、反转、扭亏阶段公司
  + 限制原因：增长率预测易出错，不适用于负增长
+ *EV/Sales*、*EV/FCF*等针对特定指标。


另类估值法
----

+ 实物期权法：评估不确定性高、具分阶段决策权的项目（如研发管线、矿产勘探）。
+ 清算价值法：评估破产或拆分出售情景下的资产变现净值。
+ 分类加总估值法（SOTP）：对多元化业务集团各板块分别估值后加总。



行业适用性
====

对不同行业的公司进行估值，核心挑战在于：行业商业模式、资产结构、增长驱动因素和风险特征不同，导致适用的估值指标和估值逻辑存在显著差异。

金融行业
----

+ 银行、保险、券商
+ 核心逻辑：资产和负债主要由金融工具（贷款、债券、衍生品等）构成，账面价值与市场价值关联度高；利润受利率、信用周期和监管资本约束影响大。
+ 常用估值指标/方法：
  + 银行：市净率（P/B）、股息率、剩余收益模型（RIM）。
  + 保险：内含价值（EV）倍数（P/EV）、新业务价值（NBV）倍数。
  + 券商：市净率（P/B）、市盈率（P/E，牛熊市波动剧烈）。
+ 关键注意事项：关注资产质量（如不良贷款率）、偿付能力充足率、净息差（NIM）；避免使用P/E单独估值银行（利润波动大）。


科技行业
----

+ 软件、互联网、硬件
+ 核心逻辑：高增长、轻资产、前期研发投入大，利润可能为负或微薄；用户、流量、技术壁垒和生态价值是关键。
+ 常用估值指标/方法：
  + 早期/亏损公司：市销率（P/S）、企业价值/销售收入（EV/Sales）、企业价值/用户数或流量指标（EV/DAU、EV/MAU）。
  + 成熟盈利公司：市盈率（P/E）、市盈率相对盈利增长比率（PEG）。
  + 硬件/设备：P/E、EV/EBITDA（考虑折旧和资本开支）。
+ 关键注意事项：区分经常性收入（SaaS订阅）与一次性收入；关注毛利率、客户获取成本（CAC）、客户生命周期价值（LTV）；警惕技术迭代风险。


消费行业
----

+ 必选消费、可选消费、零售
+ 核心逻辑：品牌、渠道、定价权和现金流稳定性差异大。必选消费防御性强，可选消费周期性强。
+ 常用估值指标/方法：
  + 稳定增长消费品：市盈率（P/E）、股息折现模型（DDM）、自由现金流折现（DCF）。
  + 高成长消费品（如新零售、潮玩）：PEG、EV/EBITDA、市销率（P/S）。
  + 零售/商超：企业价值/息税折旧摊销前利润（EV/EBITDA）、市销率（P/S，关注周转率）。
+ 关键注意事项：重视品牌护城河、渠道库存、同店销售增长率；对高杠杆零售商需看EV/EBITDA而非P/E。


周期行业
----

+ 能源、化工、有色金属、航运、汽车制造
+ 核心逻辑：盈利随宏观经济、供需关系和商品价格剧烈波动，顶峰与低谷利润差异极大。
+ 常用估值指标/方法：
  + 全周期平均利润法（Normalized P/E）：基于一个完整周期的平均利润计算P/E。
  + 企业价值/产能指标：EV/储量（能源）、EV/吨产能（化工/有色）。
  + 市净率（P/B）：当行业处于低谷亏损时，P/B提供安全垫参考。
  + 资产重置成本法（适用于资源型）。
+ 关键注意事项：避免在周期顶峰使用当期利润计算P/E；关注产能利用率、库存周期、大宗商品价格趋势。


基础设施与公用事业
----

+ 电力、水务、公路、铁路、电信
+ 核心逻辑：受监管或合同约束，现金流稳定可预测，增长慢但股息高，类似于债券。
+ 常用估值指标/方法：
  + 股息折现模型（DDM）或现金流折现（DCF）。
  + 企业价值/监管资产基数（EV/RAB，适用于受监管公用事业）。
  + 股息率（Dividend Yield）、市净率（P/B）。
  + 股权自由现金流（FCFE）倍数。
+ 关键注意事项：关注监管政策变动（如电价、过路费）、资产负债率、特许经营权期限；P/E可能因高折旧而不稳定。


房地产行业
----

+ 开发、REITs
+ 核心逻辑：重资产、高负债、收入依赖项目销售或租金。
+ 常用估值指标/方法：
  + 房地产开发：净资产价值（NAV）折价/溢价（NAV = 项目可售货值折现后减负债），常用NAV vs. 市值。
  + REITs（房地产投资信托基金）：价格/运营现金流（P/FFO）、价格/调整后运营现金流（P/AFFO）、股息率。
+ 关键注意事项：关注土地储备、去化率、融资成本、出租率及租金增长；对开发商慎用P/E（收入确认滞后）。


医疗健康
----

+ 制药、生物科技、医疗器械、服务
+ 核心逻辑：研发周期长、监管严格、专利壁垒高；生物科技公司早期无收入。
+ 常用估值指标/方法：
  + 大型制药/器械：P/E、PEG、现金流折现（DCF，对重磅药管线）。
  + 生物科技（未盈利）：风险调整后的净现值（rNPV）、研发管线价值（Pipeline value）、每股现金含量。
  + 医疗服务（医院、诊所）：EV/床位、EV/诊所、P/E。
+ 关键注意事项：区分仿制药与创新药；关注专利到期风险、临床试验成功率、监管审批进度。



实操原则
====

1. 多指标交叉验证：不依赖单一估值倍数，尤其对高波动或复杂商业模式的行业（如科技+周期混合）。
2. 相对估值与绝对估值结合：P/E、P/B等相对估值用于同业比较；DCF、NAV等绝对估值用于判断内在价值安全边际。
3. 注意行业生命周期阶段：初创期重用户/管线（P/S、rNPV），成长期重增长（PEG），成熟期重现金流（P/E、股息率），衰退期重资产清算价值。
4. 区分会计差异：不同行业折旧政策（如航空、公路）、研发资本化处理（科技、制药）会扭曲P/E和EBITDA，需手动调整。
5. 考虑地域与宏观因素：同一行业在不同国家（如中国 vs 美国银行）受利率、监管、竞争格局影响，可比估值需调整折价/溢价。

在实际分析中，应首先判断公司属于*成长型、价值型、周期型还是防御型*，再根据其商业模式选择2-3种主要估值方法，并列出该行业特有的非财务指标（如用户数、储量、管线进展、监管资产基数）作为估值修正项。
