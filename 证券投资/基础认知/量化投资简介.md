量化投资并非单一技术，而是一套用数学模型替代主观判断的系统化投资体系。用一句话概括：量化投资就是“找规律、写规则、让机器执行”。但在实际顶级机构（如文艺复兴、Two Sigma）中，它已经演化成了一门融合数学、计算机科学与金融工程的精密科学。

下面为你拆解量化投资的标准作业流程（Pipeline）、主流策略类型以及现实中的残酷门槛。

标准作业流程
====

数据获取与清洗
----

+ 脏活累活，占70%精力
+ 数据源：不仅包括价量数据（OHLCV），还包括财报、研报、舆情（推特/新闻）、另类数据（卫星图像、信用卡刷卡记录、航运数据）。
+ 核心难点：幸存者偏差（删除退市股票导致回测虚高）和前视偏差（不小心用了未来数据）。清洗质量直接决定策略生死。


因子挖掘与特征工程
----

+ 寻找“Alpha”
+ 将原始数据转化为预测信号。例如，不是直接用“股价”，而是用“过去20天涨幅”、“市盈率分位数”或“机构持仓变化”。
+ 现状：传统人工挖因子（如ROE、波动率）已红海化，现在顶级机构多用遗传算法或深度学习自动生成非线性因子。


策略建模与回测
----

+ 纸上谈兵
+ 将因子输入模型（如LSTM、XGBoost或线性回归），输出预测（涨跌概率或预期收益率）。
+ 回测：在历史数据上模拟交易。关键陷阱：过拟合（策略只适合某段历史）、滑点（成交价差）和手续费。业内常说：“回测是天堂，实盘是地狱。”


执行与风控
----

+ 从模型到真金白银
+ 算法交易：大单拆小单，利用TWAP/VWAP（时间/成交量加权平均价格）减少对市场的冲击。
+ 风控：设置仓位上限、行业暴露、VaR（风险价值）止损。风控是量化基金的命脉，模型亏损时，不是优化模型，而是先砍仓位。



主流量化策略流派
====

高频做市/套利
----

+ 持仓周期：微秒级至几分钟。
+ 核心逻辑：不预测价格方向，而是赚取买卖价差、交易所返佣或捕捉订单流短暂失衡的机会。例如，同时挂出买入和卖出限价单，等待成交。
+ 核心竞争力：纯粹的技术与速度竞赛。需要FPGA（现场可编程门阵列）硬件加速、微波传输或接近交易所机房部署，以将网络延迟压缩到极致。
+ 典型代表：Citadel Securities、Virtu Financial 等流动性提供商。


统计套利/因子轮动
----

+ 持仓周期：数天至数周的中频交易。
+ 核心逻辑：利用数百个量价、基本面或情绪因子（如动量、价值、质量、低波动率）对全市场股票进行综合打分。通过做多高分组、做空低分组，构建市场中性的投资组合，赚取股票间相对定价回归或趋势延续的收益。
+ 核心竞争力：因子挖掘的深度与组合优化的精细度。经典框架多基于Barra多因子风险模型，强调风险敞口的精确控制。
+ 典型代表：大部分传统多空量化对冲基金（如AQR资本管理公司）。


事件驱动/另类数据
----

+ 持仓周期：数周至数个季度的中长周期。
+ 核心逻辑：超越常规财报和价量数据，通过爬取、购买或分析非结构化数据（如卫星图像监测停车场车流、信用卡刷卡记录、航运船舶轨迹、社交媒体舆情），比市场主流更快一步预判公司基本面或宏观指标的变动趋势。
+ 核心竞争力：获取独特数据源的能力及自然语言处理（NLP）或图像识别技术。该策略更像“基本面量化”，持仓相对集中。
+ 典型代表：文艺复兴科技的大奖章基金（高频+中频混合）、Two Sigma 的部分另类数据团队。



与主观投资的区别
====

+ 主观投资（巴菲特）：研究“为什么”（这家公司护城河深吗？），深度持有少数几只股。
+ 量化投资（西蒙斯）：研究“是什么”（历史上这种情况涨的概率是65%吗？），广度持股（可能持有上千只股票），单笔胜率略高于50%，靠大数定律稳定盈利。



残酷的现实
====

如果你是想入行或自己尝试，有几个门槛必须知道：

1.  算力军备竞赛：顶级私募的GPU集群规模不亚于大厂AI实验室。
2.  有效性衰减：一个因子被发现后，超额收益会迅速衰减。策略需要持续迭代，没有“一招鲜”。
3.  非线性过拟合：现在很多新手用深度学习，回测曲线完美，实盘却大幅亏损，因为模型捕捉的是噪声而非信号。
4.  极端风险：量化模型基于历史，无法预测“黑天鹅”（如2020年原油负油价、2024年日经熔断）。所有量化模型在流动性枯竭时都会失效。



建议的路径
====

+ 工具：放弃通达信，转学 Python（Pandas、NumPy）和 SQL。
+ 免费数据源：从 Tushare 或 Yahoo Finance 获取A股/美股数据。
+ 入门策略：从最简单的双均线金叉死叉开始，然后升级到多因子打分模型，最后尝试机器学习（XGBoost）。

切记：实盘之前，模拟盘跑至少6个月，且要计算夏普比率和最大回撤。必要时要考虑历史上的极端行情（如2015年股灾、2020年熔断），保留人工干预权限， 而非只看年化收益。
