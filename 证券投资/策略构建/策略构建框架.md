建立股票交易策略，本质上是一个“认识市场”与“认识自我”不断对齐的过程。以下是一个从零开始的系统框架，按逻辑顺序排列：

自我认知与定位
====

+ 明确资金属性：区分这笔钱是长期闲置资金、中期储蓄还是短期流动资金。资金的可承受波动幅度决定了策略的容错空间。
+ 界定收益预期：设定一个理性的年化目标（如跑赢通胀、对标GDP增速或沪深300指数）。预期收益越高，需要承受的波动和回撤通常越大。
+ 评估风险承受力：诚实回答自己能接受本金最大亏损多少百分比（例如-20%或-30%）而不影响睡眠和决策。
+ 确定时间精力投入：每天能投入多少时间看盘？是专职交易、中午抽空还是仅盘后复盘？这直接决定了你适合盯盘交易还是中长线布局。


核心理念与信仰
====

+ 选择分析流派：决定自己主要依赖基本面分析（买好公司）、技术面分析（看图择时），还是量化统计（基于概率）。选适合自己的，切忌什么都想要，容易混乱。
+ 确立盈利来源：想清楚你赚的是企业成长的钱（伴随公司增值），还是市场情绪的钱（低买高卖的差价），或是流动性的钱（资金涌入时的溢价）。
+ 定义有效市场假说：承认市场是否总是有效。如果认为市场经常犯错，就侧重逆向投资；如果认为市场永远正确，就侧重顺势而为。


行情研判与选股
====

+ 自上而下或自下而上：先选景气行业再选个股，还是直接精研个股不管大盘。
+ 选股逻辑（定量指标）：
  + 基本面：市盈率、市净率、净资产收益率、净利润增速、现金流状况。
  + 技术面：均线多头排列、成交量异常放大、突破关键阻力位、相对强弱指标位置。
+ 选股逻辑（定性指标）：管理层能力、行业护城河、产品竞争力、政策支持力度。
+ 建立备选池：持续跟踪20-30只股票，而不是临时起意随机买入。


买卖时机与执行
====

+ 入场信号：什么条件满足时才会买入？是财报超预期、站上60日均线，还是估值落入历史低分位数？必须量化。
+ 出场信号（止盈）：上涨多少开始分批卖出？是移动止盈（回撤多少离场），还是目标价位到达即清仓？
+ 出场信号（止损）：这是最重要的一环。设定硬性止损线（如-8%无条件离场），还是逻辑止损（业绩证伪即离场）？
+ 交易频率：明确自己是日内交易、短线波段（数天）、中线趋势（数周至数月）还是长线持有（数年）。频率不同，策略截然不同。


仓位管理与资金分配
====

+ 单只个股上限：单只股票占总资产比例不超过多少（如20%）？避免黑天鹅导致毁灭性打击。
+ 分批建仓规则：是一次性全仓，还是分3批（如下跌5%加仓一次）？加仓间隔和比例需要提前设定。
+ 行业分散程度：同一板块的持仓总和上限是多少？避免持仓表面分散但实际都押注在同一个行业逻辑上。
+ 现金管理：长期保持几成仓位？是否永远不满仓，保留随时抄底的备用金？


交易记录与日志
====

+ 记录每笔交易：不仅记录买入价、卖出价、盈亏，更要记录当时买入和卖出的理由。
+ 记录心理状态：下单时是贪婪、恐慌，还是平静理性？情绪是决策质量的重要反向指标。
+ 保存截图：对于关键买卖点，截取当时的K线图和成交量，方便日后回看复盘。


复盘、评估与迭代
====

+ 固定复盘周期：每周做一次小复盘（回顾周内操作），每月做一次中复盘（统计胜率和盈亏比），每季度做大复盘（检视整体策略有效性）。
+ 统计核心指标：计算自己的胜率（盈利交易次数占比）和盈亏比（平均盈利金额/平均亏损金额）。两者往往此消彼长，需找到平衡点。
+ 归因分析：区分盈利是源于市场整体上涨（Beta），还是选股/择时能力（Alpha）。如果剔除市场上涨后自己还在亏钱，说明策略本身有问题。
+ 迭代优化：每次只修改一个变量（如仅调整止盈线，或仅更换一个指标），测试一段时间后再改下一个，避免同时修改多处导致无法归因。


纪律与心态控制
====

+ 制定交易清单：将买入和卖出的所有条件做成一个Checklist，每次操作前逐条打钩，避免冲动交易。
+ 强制冷静期：如果连续亏损3笔交易，强制停止交易1-3天；如果单月回撤超过预设阈值，强制减仓至半仓以下。
+ 接受不完美：接受卖飞和买早是常态，不以单次输赢论英雄，只关注策略长期概率是否站在自己这边。
+ 生活与交易隔离：确保交易决策不依赖“回本”的执念，不用生活必须资金入市。

最后的建议：在投入真金白银前，先用模拟盘或小资金（如总资产的5%）运行上述策略至少3个月。这段时间不是为了赚多少钱，而是为了暴露你在执行环节的漏洞，以及验证这套流程是否真的符合你的日常作息和心理承受力。策略是活出来的，不是写出来的。
