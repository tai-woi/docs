顶和底是*区域*而非精确的点，且周期不同，顶底的定义完全不同。

这里把需要观测的对象拆解成三个层级（大盘、行业、个股），每个层级用 5 个核心维度来交叉验证。记住：*单一信号是噪音，共振信号才是趋势拐点。*

大盘指数
====

大盘指数是系统性顶底，*80% 的个股服从大盘*。

+ 估值维度（最硬核）：看全市场PE/PB的历史百分位（如中证全指）。顶部常见*股债性价比（ERP）* 跌破历史均值减1倍标准差；底部则相反，ERP升至+2倍标准差以上，且破净股占比超10%（A股历史大底特征）。
+ 资金维度（最敏感）：顶部现天量天价（量先价行，爆量滞涨是危险信号）；底部现地量地价（缩量到峰值30%-40%后，出现连续放量阳线）。
+ 情绪维度（最极端）：顶部新基金一日售罄、融资余额达阶段性峰值；底部则明星基金经理被群嘲、新发基金失败，且新股首日破发成常态。
+ 技术维度（最直观）：月线级别出现长上影线或跳空缺口被回补；底部则出现月线长下影线（定海神针）或周线底背离（价格新低，MACD不创新低）。
+ 政策维度（A股特色）：顶部查违规资金、喊话降温；底部则密集出台利好（降准降息）、限制IPO、国家队ETF放量买入。


行业板块
====

行业板块是结构性顶底，板块见顶常早于大盘，*逻辑证伪*是核心。

+ 景气度维度：顶部时业绩增速最陡峭，但股价不涨（利润兑现）；底部时最坏预期全部打满，库存周期见底，龙头公司开始提价。
+ 机构持仓维度：顶部对应持仓占比历史极值（如消费板块超配30%以上），且券商集体猛推；底部则是低配或清仓，行业研报无人问津。
+ 龙头比价维度：顶部龙二龙三拼命补涨，甚至垃圾股暴涨（鸡犬升天）；底部龙头率先止跌横盘，不再创新低，且跌幅远小于跟风股。
+ 政策催化维度：顶部政策利好边际减弱（利好出尽）；底部则迎来困境反转政策（如供给侧改革、行业补贴落地）。


个股
====

个股是交易层面的顶底，个股受庄家/主力影响大，注重*量价行为*。

+ 主力行为维度：顶部出现高位放量滞涨（换手率超20%但股价横盘），且出现尾盘拉抬次日低开的诱多；底部出现无量空跌后，突然放量脉冲式拉升（主力试盘）。
+ 筹码维度：顶部低位筹码峰快速上移并消失（主力派发完毕）；底部高位套牢筹码开始割肉下移，在低位形成新的密集峰。
+ 基本面维度：顶部常伴随高管减持、大额分红（变相套现）；底部则出现回购注销、高管增持且承诺不减持。
+ 技术破局：顶部跌破上升趋势线且反抽不过前高；底部突破下降趋势线且回踩不破前低（形成N字结构）。
+ 散户情绪（反身性）：顶部时股吧里全是晒单赚钱，底部则全是咒骂销户，且利空不跌（底部特征）、利好不涨（顶部特征）。


3条“避坑”原则
====

1. *不猜最后一根K线*：只做右侧确认。顶部等跌破20日均线且拐头向下再离场，底部等站上60日均线且放量再进场。
2. *三周期共振*：日线级别信号看周线趋势，如果周线是下跌中继，日线底背离可能失效。
3. *终极检验*：底部是“涨出来的”（必须用阳线确认），顶部是“跌出来的”（必须用阴线确认）。永远不要因为“跌多了”就抄底。
