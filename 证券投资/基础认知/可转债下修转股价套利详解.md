这个手法是A股市场上产业资本（大股东）利用规则进行的经典财技。在股权视角下，它常被老股民斥为“合法稀释散户的流氓机制”。

制造不得不修的绝境
====

+ 背景：可转债有“回售条款”，通常规定“如果正股价格在连续30个交易日中，有15天低于当期转股价的70%”，持有人有权把债券按面值（100元）+利息卖回给上市公司。
+ 痛点：上市公司发债募资是为了花钱搞项目，最怕“回售”。因为一旦触发，公司必须掏出几十亿现金还债，这会直接抽干公司现金流。
+ 股价下跌：此时正股确实在跌，且跌破了回售触发价。上市公司面临巨大的资金压力。


下修转股价
====

+ 操作：上市公司董事会提议，将转股价向下大幅修正。例如，原转股价是10元，现在正股跌到6元，公司直接把转股价下修到6.5元。
+ 账本变化（关键）：
  + 下修前，每张100元的转债可转股：100 / 10 = 10股。
  + 下修后，每张100元的转债可转股：100 / 6.5 ≈ 15.38股。
  + 结论：一张债券能兑换的股票数量瞬间增加了53.8%。


潜伏与收割
====

1. 提前潜伏：在上市公司发布下修公告前（或过程中），产业资本（通常是大股东的一致行动人、关联私募）利用信息优势或精准判断，在低位大举买入可转债。此时转债价格往往因正股下跌而跌至100-105元附近（债底保护）。
2. 下修引爆：公告下修后，由于转股价值飙升，可转债价格通常会在次日跳涨至118-125元区间。
3. 转股卖出（不掏钱）：产业资本在跳涨后并不卖出债券（因为卖出赚的是差价，要交税且量太大没人接盘），而是申请将债券全部转为股票。
4. 二级市场抛售：转股后的第二天（T+1），产业资本将新增的股票在二级市场以6.5元左右的价格抛售给散户。

关键结果：产业资本当初买债券花了1亿，转股后卖出股票拿回了约1.2亿。这1.2亿不是从自己兜里掏的，而是从二级市场接盘散户兜里掏的。公司没出一分钱现金，就还掉了1亿的债券债务。


摊薄散户权益
====

这就完成了“不掏钱的定向增发”。之所以说摊薄了散户权益，体现在两个维度：

+ EPS（每股收益）稀释：总股本变大了。原先公司总股本1亿股，下修转股后突然多出1500万股。公司一年的净利润还是1个亿，每股收益从1.00元被动降到了0.87元。散户持有的每一股对应的公司资产和利润都缩水了。
+ 股价抛压：产业资本转股后的成本极低（约等于债底价格），他们的抛售是不计成本的。在正股本就下跌的脆弱市场中，这1500万新增股票涌入，会进一步砸低股价，导致持有正股的散户账面亏损扩大。


隐蔽陷阱与博弈焦点
====

+ *大股东为何愿意“主动”下修？* 表面看大股东股权也被稀释。但深层次看，大股东往往早就通过质押、减持等方式把风险转移了，或者他们手里也持有大量转债（这是合规的）。下修等于用全体的“股权池”去填“债务窟窿”，而大股东可以通过参与此次套利，弥补股权稀释的损失，甚至大赚一笔。
+ *真正的底线在哪里？* 监管规定下修后的转股价不得低于每股净资产（BVPS）。如果正股价格已经跌破净资产，下修空间就被封死了，这个手法就玩不转。
+ *散户如何识别与应对？*
   + 当你持有的正股公司发行了可转债，且正股价长期低迷时，不要去博反弹，因为下修预期一旦兑现，转债吃肉，正股挨打。
   + 如果在下跌途中，你发现转债成交量突然异常放大，且价格坚挺（不随正股下跌），大概率是产业资本或机构在悄悄收集筹码，等待下修公告的“发令枪”。

总结一句大白话：这本质上是用全体股东（尤其是散户）的股权价值作为“赎金”，去拯救上市公司必须偿还的刚性债务（转债回售）。产业资本充当了“中间商”，利用信息与规则差，将本应由大股东承担的偿债成本，成功转嫁给了二级市场的流动性。这确实是合法的，但对散户而言，是极其残酷的财富转移。如果你持有此类正股，在公告下修前，最好的防守就是远离。
