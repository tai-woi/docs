开户条件
====

+ 年龄：必须年满18周岁，具有完全民事行为能力。70岁以上老人也能开户，但部分券商可能需要临柜办理或做风险评测。
+ 身份：持有有效的二代居民身份证，且必须是中国大陆居民（港澳台及外籍人士需满足特定条件）。
+ 账户：需要有一张本人名下的一类银行卡（借记卡），用于银证转账。
+ 结论：0资金要求，不开通权限的话，账户里没钱也能开。
+ 其它：*银证转账有固定时间*（交易日9：00-16：00），且三方存管必须是你本人银行卡，不能用信用卡或他人的卡。



交易门槛
====

交易板块
----

+ A股主板（60开头、00开头的股票），无资金要求，开户即买
+ *以下板块需2年交易经验*
+ 创业板（30开头），开通前20个交易日日均资产达10万元
+ 科创板（688开头），开通前20个交易日日均资产达50万元
+ 北交所（8开头），需50万元
+ 可转债，无资金门槛
+ 融资融券（加杠杆），需50万元


交易规则
----

+ *交易时间（工作日）*：上午 9：30 - 11：30，下午 13：00 - 15：00。其中9：15 - 9：25为集合竞价时间（9：20-9：25不可撤单）。
+ *交易机制（T+1）*：当天买入的股票，下一个交易日才能卖出。但资金方面，当天卖出股票回笼的资金当天可以用来买股票，只是提现到银行卡需等到下一个交易日。
+ *涨跌幅限制*：主板股票上市首日不设限，之后常规±10%；ST/*ST股（风险警示股）为±5%；科创板/创业板为±20%；北交所为±30%。
+ *申报单位（手）*：买入必须是100股（1手）的整数倍；卖出时，不足100股的部分必须一次性全部卖出。
+ *价格优先、时间优先*：成交时，出价高的买单优先于出价低的买单，出价低的卖单优先于出价高的卖单；价格相同时，先申报的先成交。
+ *委托价格限制*：下单时，买入价不能高于“卖一”价的102%（主板），卖出价不能低于“买一”价的98%，超出范围会成为废单（科创板/创业板规则略有不同）。
+ *停牌规则*：新股上市首日或盘中涨幅过大，可能触发临时停牌（通常停牌10分钟或持续到14：57）。


打新规则
----

核心逻辑是“先有市值，才有资格”，且“中签像抽奖”。以下是主板打新最核心的几条规则：

+ 门票条件（市值配售）：申购前连续22个交易日（约1个月），你账户持有的股票日均市值必须达到1万元以上。注意，只有股票市值算数，现金不算。
+ 额度计算（沪深分开算）：上海和深圳的市值不通用，要分开计算。通常每5000元或1万元市值对应1个申购单位（具体看板块），市值越大，给你的抽签号码（配号）就越多。
+ 申购操作（免费抽签）：在交易软件里点击“新股申购”就行，不需要提前交钱，只有中签后才需要缴款。
+ 中签与缴款（关键一步）：申购后第2个交易日（T+2日）公布中签结果。如果幸运中签，必须在当天下午16：00前确保银行卡或账户里有足够的钱，系统会自动扣款。如果忘记缴款，就算弃购。
+ 弃购惩罚（务必注意）：12个月内如果累计弃购3次，会被拉入“黑名单”，180天内禁止你再打新。

特别提醒（避坑）：

+ 多账户无效：同一个身份证名下的所有账户，对同一只新股只有第一次申购有效，重复申购无效。
+ 板块权限：想打创业板或科创板的新股，必须提前开通对应的交易权限（比如科创板需要50万资产和2年经验）。



交易费用
====

印花税
----

+ 费率：成交金额的 0.05%（万分之五）
+ 规则：只在卖出时收，买入时不收
+ 这是大头，国家收。比如你卖出1万元股票，印花税就是5元


过户费
----

+ 费率：成交金额的 0.001%（十万分之一）。
+ 规则：买卖双向收取
+ 交易所收，金额很小，1万元买卖只收0.1元，基本可以忽略

交易佣金
----

+ 费率：官方最高不超过 0.3%（千分之三），但现在的行情，默认开户通常是 0.025%（万分之2.5）
+ 规则：买卖双向收取。注意！ 很多券商有“最低5元”的规矩，也就是单笔佣金算下来不足5元的，按5元收
+ 券商收，这是你唯一可以和券商讨价还价的部分


券商选择
====

选择头部券商
----

+ 优先选头部大券商（如中信、华泰、国泰君安、银河等）+ 优秀的互联网券商（如东方财富、同花顺合作的券商）。
+ 大券商通道稳，互联网券商体验好。别选太小的地方性券商，怕系统关键时刻掉链子。
+ 对于新人，界面简洁、挂单成交快、行情不卡顿比什么都重要。

咨询免五
----

+ 避坑：如果你资金小，每次只买几千块，按万分之2.5算佣金才1-2元，但如果券商收“最低5元”，那你实际费率就被强行拉高到千分之几，成本瞬间翻几倍。
+ 建议：直接打电话给客服或找客户经理，问清楚“我的账户佣金有没有最低5元限制”。 现在很多互联网券商已经取消了这个限制，按实际比例收。这对小资金新人非常友好。

咨询费率可调
----

+ 现在的行情，万分之1.5到万分之2是比较合理的佣金水平。如果你直接在APP上默认开户，通常是万分之2.5
+ 技巧：不要直接在APP上点“立即开户”。先去找个客户经理（线上就能找），跟他说“我准备入金XX万，佣金能给到万1.5吗？”
+ 万一免五也不是没可能。


一点建议
----

*费用是“次要矛盾”，交易习惯才是“主要矛盾”。*

很多新人亏钱，不是因为佣金高，而是因为频繁做T（日内短线）。印花税只在卖出时收，你每多交易一次，就多交一次印花税和佣金。如果每周交易2次，一年下来交易成本可能吃掉本金的5%-10%。

所以最终建议是：选一个取消“最低5元”、佣金在万1.5左右的头部券商，然后忘记手续费这件事，把精力放在选好公司和拿得住上面。
