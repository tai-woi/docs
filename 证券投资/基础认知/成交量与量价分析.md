成交量
====

基本概念
----

成交量是衡量市场活跃度、资金流向与交易真实性的核心指标。

+ 成交量：某一时段内成交的证券总数量（股数/手数）或总金额。
+ 成交额：成交量 × 成交均价，更能反映资金规模（尤其适合跨股票/指数比较）。
+ 换手率：成交量 ÷ 流通股本 × 100%，衡量筹码交换频率。
+ 内盘：主动性卖盘数量（卖方主动降价成交）。
+ 外盘：主动性买盘数量（买方主动提价成交）。
+ 盘口：分时成交明细中的逐笔交易数据。


核心维度
----

### 绝对水平

+ 放量：成交量显著高于近期均值（如高于20日均量50%以上）。
+ 缩量：成交量显著低于近期均值（如低于20日均量30%以上）。
+ 地量：长期缩量至阶段最低水平。
+ 天量：历史性或阶段性的极端大量。

### 相对位置

+ 低位区域：股价处于长期下跌后的底部区间。
+ 中位区域：横盘震荡或趋势中途。
+ 高位区域：股价处于长期上涨后的顶部区间。

### 时间持续性

+ 温和放量：连续多日逐步放大。
+ 急速放量：单日或两日内急剧扩大。
+ 连续缩量：持续多日逐渐缩小。
+ 间歇性放量：时而放量时而缩量，无规律。

### 资金行为特征

+ 建仓型放量：低位温和放量，阳线多阴线少。
+ 对倒型放量：盘中突然脉冲放量，挂单异常（大单托底+大单卖出）。
+ 出货型放量：高位放量但股价滞涨或阴线收盘。
+ 恐慌型放量：利空或破位时涌出割肉盘，长阴带巨量。


常见形态
----

### 递增形态

+ 特征：连续3个以上交易日成交量逐级放大。
+ 含义：场外资金有序入场，趋势动能增强（上升趋势中为良性，下降趋势中需警惕接盘）。

### 递减形态

+ 特征：连续3个以上交易日成交量逐级缩小。
+ 含义：交投降温，原有趋势动能衰减（易引发短期转折或停顿）。

### 三角形态

+ 特征：成交量先递增后递减，呈对称三角形变化。
+ 含义：资金先集中后观望，通常预示变盘（方向取决于后续放量突破方向）。

### 圆弧形态

+ 特征：成交量由小渐大，达到峰值后再渐小，呈圆弧状。
+ 含义：一轮完整的多空转换过程，圆弧左侧对应趋势启动，右侧对应趋势衰竭。

### 散兵坑形态

+ 特征：成交量从缩量到极致（地量），随后温和放大。
+ 含义：洗盘结束、筹码沉淀，后期易出现趋势性行情（往往出现在调整末端）。

### 脉冲形态

+ 特征：单日或两日突然放出巨量，随后迅速恢复原量级。
+ 含义：突发消息刺激或主力试盘/对倒，不具备持续性，易形成短期高点或低点。

### 不规则形态

+ 特征：成交量忽大忽小，无稳定规律。
+ 含义：市场分歧极大或缺乏主导资金，趋势稳定性差，多出现在震荡市或庄股后期。


误区和注意点
----

### 误区

+ ~~放量一定代表上涨可靠：~~正解：高位放量滞涨往往是出货信号。
+ ~~缩量一定代表无人关注：~~正解：上涨后缩量整理代表筹码锁定良好，属于强势调整。
+ ~~天量之后必然有天价：~~正解：天量后若无法维持放量，极易形成历史顶部（尤其是阴线天量）。
+ ~~外盘大于内盘就是真买入：~~正解：主力可通过“大单拆小单”方式制造外盘大的假象，需结合成交明细中的大单方向。

### 注意点

+ *流通股本差异：* 大盘股与中小盘股的绝对成交量不可直接比较，应使用换手率或成交额相对值。
+ *不同市场的量级特征：* 期货、债券、ETF的成交量分析逻辑与股票有差异（如期货需区分开仓量与平仓量）。
+ *停牌复牌后的成交量失真：* 复牌首日无涨跌幅限制或补涨/补跌需求，其成交量不具有常规参考意义。
+ *突发消息导致的异常量：* 重大政策、业绩预告、黑天鹅事件造成的瞬间放量，需要等待消息消化后再分析。


实践步骤
----

1. 观察当前成交量相对于20日或60日均量的位置（判断放量/缩量等级）。
2. 确认股价所处的相对位置（高位/低位/中位）及趋势方向。
3. 识别最近5-10个交易日的成交量形态（递增、递减、散兵坑等）。
4. 分析盘中大单主动性（外盘/内盘差值，分时成交中的万手单方向）。
5. 结合换手率判断筹码活跃程度（3%以下清淡，3%-7%活跃，7%以上异常）。
6. 排除因除权、停牌、新股上市等非市场因素导致的成交量异常。
7. 综合得出资金行为结论（如：低位温和放量→建仓；高位不规则放量→出货）。



量价分析
====

底部反弹与突破
----

+ 价格行为：长期下跌后，低位放量反弹（量显著高于均量）
  + 买入信号：较强买入（轻仓试多）
  + 卖出/减仓信号：无
  + 说明：可能主力建仓或反弹开始，需设止损
+ 价格行为：长期下跌后，低位缩量反弹
  + 买入信号：无（观望）
  + 卖出/减仓信号：无
  + 说明：买盘不足，反弹大概率失败
+ 价格行为：底部横盘后，放量上涨突破阻力位
  + 买入信号：强烈买入（可跟进）
  + 卖出/减仓信号：无
  + 说明：真突破概率高
+ 价格行为：底部横盘后，无量上涨突破阻力位
  + 买入信号：不建议买入
  + 卖出/减仓信号：无
  + 说明：假突破风险大，等待补量


持续上涨过程
----

+ 价格行为：持续上涨中，价涨量增
  + 买入信号：持有或继续买入（趋势健康）
  + 卖出/减仓信号：无
  + 说明：买盘积极，上涨真实
+ 价格行为：持续上涨中，价涨量平
  + 买入信号：谨慎持有（不加仓）
  + 卖出/减仓信号：无
  + 说明：动能一般，观察是否补量
+ 价格行为：持续上涨中，价涨量缩
  + 买入信号：无
  + 卖出/减仓信号：较强卖出（分批止盈）
  + 说明：量价顶背离，上涨动能不足
+ 价格行为：持续上涨中，价平量增
  + 买入信号：无
  + 卖出/减仓信号：强烈卖出（清仓或大幅减仓）
  + 说明：高位滞涨，出货特征


回调过程
----

+ 价格行为：上涨后的回调，缩量
  + 买入信号：较强买入（回调末端低吸）
  + 卖出/减仓信号：无
  + 说明：抛压小，趋势可能延续
+ 价格行为：上涨后的回调，放量
  + 买入信号：无
  + 卖出/减仓信号：一般减仓（降低仓位）
  + 说明：抛压加重，可能转势


下跌过程
----

+ 价格行为：下跌过程中，价跌量增
  + 买入信号：无（不抄底）
  + 卖出/减仓信号：强烈卖出（离场）
  + 说明：恐慌抛售，下跌加速
+ 价格行为：下跌过程中，价跌量平
  + 买入信号：无
  + 卖出/减仓信号：一般卖出（维持看空）
  + 说明：稳定下跌，无人承接
+ 价格行为：下跌过程中，价跌量缩
  + 买入信号：无（观望）
  + 卖出/减仓信号：不新卖出（已持有则扛或止损）
  + 说明：卖盘减轻，但未止跌
+ 价格行为：长期下跌后，价跌量极度萎缩（地量）
  + 买入信号：一般买入（轻仓试探）
  + 卖出/减仓信号：无
  + 说明：卖盘枯竭，需放量阳线确认


下跌后反弹
----

+ 价格行为：下跌趋势中反弹，缩量
  + 买入信号：无
  + 卖出/减仓信号：一般卖出（反弹减仓）
  + 说明：弱势反弹，大概率续跌
+ 价格行为：下跌趋势末端反弹，放量
  + 买入信号：一般买入（试多，需确认）
  + 卖出/减仓信号：无
  + 说明：可能转势，需站上关键位


顶部区域
----

+ 价格行为：高位横盘后，价平量增
  + 买入信号：无
  + 卖出/减仓信号：强烈卖出（清仓）
  + 说明：出货明显
+ 价格行为：高位出现天量长阳
  + 买入信号：无
  + 卖出/减仓信号：强烈卖出（离场）
  + 说明：高潮或对倒出货
+ 价格行为：高位下跌后反抽，缩量
  + 买入信号：无
  + 卖出/减仓信号：强烈卖出（逃命波）
  + 说明：反抽无力，确认顶部



假突破的量价特征
====

向上假突破（多头陷阱）
----

+ 突破时的成交量特征
  + 突破关键阻力位时成交量并未显著放大，甚至低于近期均量
  + 突破当日或次日成交量快速萎缩，缺乏持续买盘跟进
  + 可能出现盘中瞬间放量拉高，但全天成交量平平
+ 突破后的价格行为特征
  + 突破后无法站稳阻力位上方，通常在1-3日内跌回原区间
  + 突破后出现长上影线、阴线或十字星等反转K线
  + 突破后次日即低开低走，无继续上攻意愿
+ 突破前的铺垫特征
  + 突破前未经历充分的横盘整理或底部换手
  + 突破前出现急促拉升，缺乏逐级放量过程
+ 与大盘或板块的对比特征
  + 大盘或板块处于下跌或震荡中，个股独自“突破”，缺乏环境配合
  + 同期同板块其他个股并未跟随突破
+ 位置特征
  + 出现在长期上涨后的高位区域
  + 出现在主力出货阶段的末段，诱多性质明显


向下假突破（空头陷阱）
----

+ 突破时的成交量特征
  + 跌破关键支撑位时成交量萎缩，未出现恐慌性抛售
  + 跌破后迅速缩量，杀跌动能枯竭
  + 可能出现快速无量下探，随后快速收回
+ 突破后的价格行为特征
  + 跌破后1-3日内迅速回升至支撑位上方
  + 跌破后出现长下影线、阳线或启明星等反转K线
  + 跌破后次日即高开高走，无持续下跌意愿
+ 突破前的铺垫特征
  + 跌破前未出现持续放量下跌，而是阴跌后突然破位
  + 跌破前已有较长时间缩量调整，抛压自然减轻
+ 与大盘或板块的对比特征
  + 大盘或板块企稳或反弹时，个股单独“破位”，刻意制造恐慌
  + 同期同板块其他个股并未跟随破位
+ 位置特征
  + 出现在长期下跌后的低位区域
  + 出现在主力建仓末期，诱空吸筹性质明显


盘整区间假突破
----

+ 成交量特征
  + 突破边界时成交量无法持续放大，单日放量后迅速萎缩
  + 突破时的成交量低于区间内前一次尝试突破时的量能
  + 突破后回踩时成交量无明显萎缩（说明抛压仍大）
+ 价格行为特征
  + 突破后无法形成趋势性延续，快速返回区间内部
  + 突破K线实体较小，或带有明显长影线
  + 突破后连续2-3日反向运行，吞噬突破幅度
+ 时间特征
  + 突破后有效站稳新价格区间的时间不足3日
  + 突破发生在交易清淡的时段（如午盘或尾盘偷袭）
+ 均线辅助特征
  + 向上假突破时，股价虽突破阻力位但中长期均线仍呈空头排列
  + 向下假突破时，股价虽跌破支撑位但中长期均线仍呈多头排列


识别与应对
----

+ 识别要点
  + 真突破通常伴随持续放量或温和递增放量，假突破多为单日异常放量或无量突破
  + 真突破后往往不会快速返回原区间，假突破则快速折返
  + 真突破会带动同板块或指数共振，假突破多为个股独立走势
  + 真突破前通常有充分蓄势，假突破常缺乏酝酿过程
+ 应对策略
  + 向上突破时不急于追高，等待突破后回踩确认有效再介入
  + 向下突破时不急于割肉，等待反抽确认有效跌破再离场
  + 设置3日或3%的过滤规则：突破后3日不返回区间，或站稳3%以上，才视为有效
  + 用收盘价判断突破，不以上影线或盘中瞬间价格为准
