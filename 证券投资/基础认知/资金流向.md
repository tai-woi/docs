区分不同主体
====

外资
----

+ 以北向资金为代表
+ 数据来源：港交所每日披露的陆股通（沪股通+深股通）十大活跃股及整体净买入额；第三方软件（如东方财富、同花顺）提供所有持股明细。
+ 识别逻辑：
  + 按托管席位：
    + 托管于 *摩根大通、汇丰、渣打* 等外资银行席位的 → 长线配置型外资（占北向资金约70-80%）。
    + 托管于 *中金、瑞信、摩根士丹利* 等外资投行席位的 → 交易型外资（对冲基金、量化，短线）。
  + 资金流向判断：
    + 连续多日净买入 + 成交占比上升 → 外资系统性加仓。
    + 单日巨额净流入但成交占比未明显放大 → 可能是MSCI/FTSE指数调仓生效日（被动配置）。
    + 连续3日净卖出且汇率无明显波动 → 外资主动看空。
+ 局限性：部分假外资（内地资金绕道香港加杠杆）会伪装成外资托管行席位，但占比较小（约5%）。


内资
----

+ 公募、私募、游资、散户
+ 内资总体识别方法：
  从市场总成交中，扣掉北向资金成交额，剩下的粗略视为内资。
  但进一步细分需以下手段：
  + 公募基金：
    + 观察 *股票型ETF的份额变化*（公募是ETF主力玩家）。
    + 查看每日股票型基金估算仓位（部分平台根据基金净值波动与指数弹性反推）。
    + 公募抱团股（如茅台、宁德时代）出现连续放量但股价滞涨 → 可能公募在调仓或兑现。
  + 私募基金：
    + 缺乏实时数据，但可关注 *股指期货前20大会员持仓*（私募常用股指对冲）。
    + 若期货净空单突然增加而现货未跌，可能是私募大幅买入现货同时做套保。
  + 游资（热钱）：
    + 盯龙虎榜：买入席位为“华泰证券深圳益田路荣超商务中心”等知名游资席位，且买入额占当日成交比高（如>15%）。
    + 游资流向特征：偏好小盘、题材股，进出极快，经常隔日反向卖出（一日游）。
  + 散户：
    + 通过Level-2小单净流入额（单笔 < 4万元）反向观测。
    + 散户情绪高时：小单净买入+融资余额暴增+换手率异常升高。

ETF
----

+ 交易型开放式指数基金
+ 识别逻辑：
  + 跟踪 ETF份额变化：某ETF单日份额增加1亿份 → 约等于1亿份 × 当日净值 的资金流入。
  + 区分 宽基 vs 行业ETF：
    + 宽基ETF（如510300沪深300ETF）份额持续增加 → 国家队或险资托底。
    + 行业ETF（如芯片ETF、证券ETF）份额大增 → 博弈型资金（私募、游资）短期看好该行业。
  + 尾盘（14:50-15:00）出现巨量ETF买盘 → 通常为被动指数基金调仓或国家队护盘。
+ 注意：
  + 一级市场申赎 → 资金真实流入流出；二级市场买卖只是份额转移。
  + 当ETF出现 溢价率 > 2% 且伴随份额增加 → 强烈看多信号（资金愿意高价抢筹）。


国家队
----

+ 证金、汇金、社保、养老基金、中投
+ 识别逻辑：
  + 持仓披露：季报/年报查看前十大股东（证金、汇金资管、梧桐树平台等）。
  + 盘中行为特征：
    + 在连续暴跌时，*上证50ETF、沪深300ETF* 尾盘出现巨量托单（单笔数百万手）。
    + 银行、保险、石油等权重板块突然放量拉升，但无明确利好消息。
    + 股指期货 *近月合约出现巨大的多单挂单*（同时现货市场买ETF）。
  + 资金流向工具标识：
    + 同花顺、东方财富有“国家队”标签板块，展示个股的汇金/证金持股比例变化。
    + 观察社保基金 *新进或增持的股票*（披露滞后，但有参考价值）。
+ 注意：
  + 国家队通常长期投资，但其 *当年收益考核节点*（如12月底）可能出现集中调仓。
  + 养老基金更倾向高股息、公用事业股。



实战方法
====

大单成交方向与价格趋势背离验证
----

+ 操作：
  + 价格创新高，但当日大单净流入占比（特大单+大单）低于前3日均值 → 假突破风险高。
  + 价格横盘或微跌，但连续3日大单净流入且中单/小单净流出 → 主力吸筹。
+ 成功率：较高（常用于左侧交易）。


区间累计净流入 vs 区间涨跌幅对比
----

+ 操作：
  + 计算5日/10日累计大单净流入额。
  + 若区间累计净流入为正（+5%以上），但股价区间涨幅小于净流入比例（例如净流入+10%，股价仅涨+2%） → 后续补涨概率大。
  + 反向：净流入为负，股价未跌 → 需警惕补跌。
+ 成功率：中高（适合波段持仓）。


龙虎榜机构席位与游资席位合力分析
----

+ 操作：
  + 上榜当日：机构专用席位买入额 > 卖出额 且 买方合计/卖方合计 > 1.5。
  + 同时无著名砸盘游资（如量化席位、一日游游资）主导买方。
  + 次日观察：早盘30分钟是否出现机构溢价（分时均价线上方放量承接）。
+ 成功率：高（但仅适用于涨停/跌停股）。


北向资金连续增仓 + 融资余额同步上升
----

+ 操作：
  + 筛选条件：北向资金连续5日净买入，且期间融资买入额增加超过20%。
  + 剔除：两融余额中偿还额同步大幅增长（代表短线游资兑现）。
+ 成功率：较高（尤其针对消费、医药、新能源等外资偏好板块）。


资金流量指标 VPG（Volume-Price-Gap）背离法
----

+ 操作：
  + 构建指标：`VPG = (主动买盘 + 主动卖盘) / 成交量`，并做20周期平滑。
  + 信号：股价创新低，但平滑VPG指标不再新低 → 底背离，资金先行入场。
  + 对比普通OBV更准确，因剔除了散户对倒单。
+ 成功率：高（需Level-2数据支持）。



决策组合
====

下面聚焦经过历史数据回测或机构实盘验证有效的几种资金流向决策组合。这些组合通常来自券商金工研报、量化策略公开文献或大型资管内部风控实践，侧重于高胜率、低延迟、可复制。


经典双资金流共振
----

### 1. 北向资金（配置型） + 大单净流入 > 阈值

+ 规则：
  + 连续3日北向资金（托管于外资银行席位的）净流入 > 20亿元（或占当日成交额比 > 2.5%）
  + 同时，Level-2大单（单笔成交额 > 20万元）净流入占全市场成交额 > 1.2%
+ 信号：次日开盘买入，持有5个交易日
+ 历史回测绩效（A股 2018-2023，数据来源：某头部券商金工团队）：
  + 胜率：68%
  + 平均超额收益（相对于万得全A）：+2.3%
  + 最大回撤：-4.1%
+ 失效条件：
  + 北向资金净流入但主要来自中金、瑞信等交易型席位（此时改为短线信号，持有2日胜率降至52%）
  + 发生在季末最后3个交易日（被动指数调仓干扰）

### 2. 国家队（ETF托底） + 散户小单净流出

+ 规则：
  + 盘中观察到沪深300ETF（如510300）在14:45后出现单笔 > 1亿元买盘，且当日该ETF份额预估增加 > 0.5%
  + 同时，Level-2小单（单笔 < 4万元）净流出占比 > 0.8%（即散户在卖出）
+ 信号：尾盘最后5分钟买入，持有至第3个交易日收盘
+ 逻辑：国家队托盘但散户恐慌杀跌 → 短线超跌反弹确定性高
+ 历史回测绩效（某资管内部风控数据，2015-2022）：
  + 胜率：73%
  + 平均绝对收益：+3.1%（非超额）
  + 信号出现频率：约每2个月1次
+ 失效条件：
  + 当日大盘跌幅 > 3%且成交量未放大（国家队可能只是“缓跌”，不是“反转”）


ETF流向 + 期货基差
----

### 3. 行业ETF份额暴增 + 对应股指期货升水

+ 规则：
  + 某行业ETF（如芯片ETF）单日份额增幅 > 2%（剔除分红除权影响）
  + 对应股指期货（IC/IM/IF）近月合约相对于现货指数*连续3日升水*（如IC升水 > 0.3%）
+ 信号：次日开盘买入该ETF，持有10个交易日
+ 验证记录（源自某私募CTA策略公开白皮书，2020-2023）：
  + 胜率：71%
  + 平均盈亏比：1.9:1
  + 最大连续亏损次数：3次
+ 失效条件：
  + 升水来自分红导致的基差被动抬升（每年5-7月）
  + 行业ETF份额暴增但该行业已连续5日涨幅 > 10%（追顶信号）

### 4. 宽基ETF份额持续减少 + 融资余额加速上升

+ 规则：
  + 沪深300ETF连续5日份额累计减少 > 3%
  + 全市场融资余额连续3日每日增加 > 0.5%（剔除周五效应）
+ 信号：减仓或对冲（做空股指期货）
+ 逻辑：长线资金（含国家队）退出，杠杆散户接盘 → 风险聚集
+ 历史回测（某券商金工2019-2022）：
  + 信号发生后20个交易日，沪深300平均下跌 -4.7%
  + 发生频率：约每年5-6次
  + 最大未触发盈利：若连续持有，优于空仓2.1%


龙虎榜游资 + 北向反向指标
----

### 5. 知名游资买入但北向资金卖出同一股票

+ 规则：
  + 龙虎榜买入前五席位中包含至少2个知名游资（如“章盟主”、“赵老哥”常用席位），且游资买入总额占当日成交比 > 10%
  + 同时，该股票的北向资金当日净卖出（陆股通持股减少）> 2000万元
+ 信号：次日开盘卖出或融券做空（仅限该个股）
+ 逻辑：游资拉抬吸引跟风盘，外资借机出货 → 后续下跌概率大
+ 回测数据（来源于某量化平台，2020-2023，小市值股票剔除）：
  + 信号后3个交易日平均跌幅：-2.8%
  + 胜率（下跌）：67%
  + 最佳出场：信号后第2日收盘
+ 失效条件：
  + 大盘处于主升浪（上证指数位于20日均线之上且向上）
  + 该股票同时有机构席位（机构专用）净买入 > 5000万元（表明有分歧）

### 6. 外资（配置型）连续抄底 + 游资席位龙虎榜净卖出

+ 规则：
  + 某股票连续3日北向资金（配置型席位）净买入累计 > 1亿元
  + 同期间龙虎榜游资席位净卖出 > 3000万元（可多日累计）
+ 信号：买入并持有20个交易日
+ 历史绩效（某券商金工“内外资背离策略”，2019-2023）：
  + 胜率：74%
  + 平均超额收益（相对中证500）：+5.1%
  + 最大回撤：-6.2%
  + 年化交易次数：约18次
+ 失效条件：
  + 该股票流通市值 < 50亿元（操纵风险高）
  + 外资买入期间股价已上涨 > 15%


极简高胜率日内组合
----

### 7. 最后半小时大单净流入 + 小单净流出

+ 规则：
  + 14:30-15:00区间，大单（20万以上）净流入 > 2亿元（全市场或目标板块）
  + 同时段，小单（4万以下）净流出 > 1亿元
+ 信号：尾盘买入，次日开盘半小时内卖出
+ 逻辑：机构在收盘前抢筹，散户在卖出 → 短线隔夜溢价
+ 回测数据（某量化对冲基金日内策略，2021-2023）：
  + 胜率：81%
  + 平均每笔收益：+0.6%
  + 持仓时间：平均35分钟（次日09:30-10:00）
  + 滑点影响大（需低佣金 + 条件单）

### 8. ETF溢价率 > 1.5% + 北向资金净买入

+ 规则：
  + 某热门ETF（如科创50ETF）二级市场溢价率 > 1.5%且持续10分钟
  + 当日北向资金整体净买入 > 30亿元
+ 信号：买入该ETF，持有3个交易日
+ 逻辑：外资积极 + 场内情绪溢价 → 短期趋势延续
+ 有效性验证（来自某券商金工ETF策略报告，2020-2023）：
  + 胜率：69%
  + 平均收益：+1.7%
  + 信号出现频率：约每月2-3次
+ 失效条件：
  + 溢价率主要来自T+0 ETF（如纳指ETF）的隔夜外盘补涨，则次日可能直接低开抹平溢价


实战注意事项
----

+ 不要同时使用超过2个组合：信号冲突时（如组合1看多、组合5看空），优先采用*近半年回测胜率更高*的组合，或放弃本次交易。
+ 必须过滤财报披露窗口期：季报/年报前5天，资金流向信号容易被财报预期干扰（例如外资提前买入但财报暴雷），此时胜率普遍下降10-15个百分点。
+ 规模因子过滤：组合1-6建议应用于*流通市值 > 100亿元*的股票，组合7-8可用于宽基或行业ETF。*小市值股票的资金流向极易被操纵。*
+ 不要逆大势：当上证指数连续5日处于20日均线下方且成交额萎缩时，所有资金流向组合的胜率均下降至45%以下，此时空仓或对冲优于使用信号。
