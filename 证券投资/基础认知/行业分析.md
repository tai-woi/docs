行业周期
====

经济周期阶段
----

1. 复苏阶段
+ 强周期性行业（工业、原材料、能源）：股价率先反弹，因需求回暖、产能利用率提升。
+ 金融行业（银行、保险）：信贷需求增加，坏账预期下降，股价回升。
+ 可选消费（汽车、家电、旅游）：消费者信心恢复，销售增长带动股价。
2. 繁荣（过热）阶段
+ 资源类行业（石油、煤炭、有色）：大宗商品涨价，利润大增，股价强劲。
+ 科技与硬件：资本开支扩张，企业盈利提升，但估值可能偏高。
+ 基础工业（钢铁、化工）：产能紧张，产品提价，股价上行。
+ 房地产：利率仍低时受益，但若央行加息则压力初现。
3. 衰退阶段
+ 防御性行业（公用事业、医疗保健、必需消费品）：需求刚性，盈利稳定，相对抗跌。
+ 电信服务：收入波动小，股息相对可靠，受资金青睐。
+ 黄金股：避险需求上升，但前提是通胀或地缘风险主导衰退。
+ 金融行业：坏账增加，利差收窄，股价承压。
4. 萧条阶段
+ 公用事业（水电燃气）：政府定价或合同制，现金流稳定，具防御价值。
+ 低成本必需消费品（低价食品、日用品）：刚需属性强，龙头公司股价相对稳健。
+ 高负债周期行业（地产、建筑、航空）：违约风险上升，股价暴跌。


典型行业的周期
----

强周期行业（随经济同向大幅波动）

+ 有色金属、煤炭、钢铁
+ 工程机械、重卡、船舶制造
+ 房地产、水泥、玻璃
+ 航空、航运、物流（但物流中含快递部分相对稳定）
+ 证券、保险（部分周期+市场情绪）

弱周期/防御性行业（经济下行时相对抗跌）

+ 食品饮料（尤其必选：米面、调味品、乳制品）
+ 医药生物（创新药略有成长属性，但整体需求刚性）
+ 公用事业（电力、水务、燃气）
+ 电信运营、高速公路
+ 教育、殡葬等超刚性服务

成长性行业（受自身创新周期影响大于经济周期）

+ 半导体（全球电子+资本开支周期，但衰退期也受挫）
+ 消费电子（新品周期主导，但经济差时换机拉长）
+ 互联网/软件（部分订阅模式抗跌，但广告收入随经济波动）
+ 新能源/电动车（政策+渗透率周期，阶段性独立于宏观）


同行业子板块的周期
----

#### 判别方法

问三个问题：

1. 该产品/服务是否可推迟使用？——可推迟则周期性强（如买车、买房、做医美）。
2. 该产品是否必须经常重复购买？——频率高则弱周期（如食品、药品、物业费）。
3. 经济差时是否存在“替代效应”（从高价转向低价）？——如果有，则低价替代品可能逆周期。

实践口诀：

+ 上游大宗 > 下游品牌（同行业内部）
+ 可选属性 > 必选属性
+ 增量消费 > 存量维护
+ 交易中介 > 持有运营

#### 影响差异举例

+ 汽车行业：整车（强周期）＞ 零部件（中周期，但随整车）＞ 汽车后市场养护（弱周期）
+ 化工行业：大宗基础化工（强周期）＞ 精细化工（中周期）＞ 农用化工（部分刚需，弱一些）
+ 零售行业：百货/奢侈品（强周期）＞ 超市/便利店（弱周期）＞ 折扣店（逆周期或抗跌）
+ 房地产行业：房地产中介服务（强周期，甚至放大波动）＞ 住宅开发（强周期，且受政策周期叠加）＞ 商业地产（写字楼/商场）（中强周期）＞ 物业管理（弱周期/防御性）


不同行业周期的特点
----

### 按驱动因素划分的行业周期类型

#### 1. 宏观经济周期（3-7年）

+ 驱动因素：GDP增速、利率、通胀、就业、信贷扩张与收缩
+ 影响范围：几乎所有行业，但敏感度不同
+ 强周期行业：工业、原材料、能源、金融、房地产、可选消费、航运
+ 弱周期行业：必需消费、医疗、公用事业、电信
+ 典型特征：行业需求随经济同向大幅波动；衰退期收入利润骤降，复苏期快速反弹
+ 股市表现：强周期行业指数波动率约为大盘的1.2-1.8倍
+ 代表行业：有色金属、工程机械、证券、房地产

#### 2. 库存周期（基钦周期，3-4年）

+ 驱动因素：企业主动补库存、被动去库存、主动去库存、被动补库存
+ 影响范围：制造业、批发零售业、中游加工行业
+ 阶段特征：
  + 被动去库存（需求回暖，库存下降）：股价开始上涨
  + 主动补库存（需求旺盛，企业主动加库存）：股价主升浪
  + 被动补库存（需求转弱，库存被动积压）：股价见顶回落
  + 主动去库存（需求萎缩，企业主动降价去库）：股价深跌
+ 典型行业：汽车、家电、化工、钢铁、电子元器件、半导体
+ 股市信号：库存增速与股价高度相关，领先或滞后约1-2个月
+ 代表行业：DRAM存储芯片、空调、螺纹钢

#### 3. 资本开支/产能周期（朱格拉周期，7-11年）

+ 驱动因素：设备更新、技术升级、固定资产投资、产能扩张与出清
+ 影响范围：重工业、制造业、基础设施相关行业
+ 典型特征：
  + 繁荣期：企业大规模扩产，设备商、工程商订单爆满
  + 衰退期：产能过剩，投资骤减，行业进入长期出清
  + 复苏期：旧设备需要更新，资本开支重启
+ 股市表现：设备类公司（如工程机械、机床、机器人）股价呈7-10年一轮的大幅波动
+ 代表行业：工程机械（挖掘机）、工业机器人、重型卡车、船舶制造、半导体设备

#### 4. 房地产周期（库兹涅茨周期，15-25年）

+ 驱动因素：人口结构、城镇化、土地供应、信贷政策、居民杠杆
+ 影响范围：房地产开发、建筑建材、家居、家电、银行信贷
+ 典型特征：
  + 上升期（10-15年）：房价持续上涨，销售面积创新高，上下游全面繁荣
  + 拐点：人口红利消失、城镇化放缓、政策收紧
  + 下行期（5-10年）：去杠杆、房企暴雷、销售萎缩、土地流拍
+ 股市表现：地产股呈长周期大顶，一旦拐点确认，可能10年不创新高
+ 代表行业：住宅开发、水泥、玻璃、家居卖场、建筑装饰

#### 5. 技术/创新周期（康德拉季耶夫周期，40-60年）

+ 驱动因素：重大技术革命（蒸汽机、电力、计算机、互联网、AI）
+ 影响范围：技术驱动的所有行业，尤其是新兴行业
+ 典型特征：
  + 萌芽期：技术不成熟，商业化前景不明，投机资金进入
  + 爆发期：技术突破，商业模式确立，需求爆发式增长
  + 狂热期：资本过度涌入，估值泡沫，竞争加剧
  + 出清期：泡沫破裂，弱者淘汰，龙头胜出
  + 成熟期：行业增速放缓，进入平稳增长
+ 股市表现：新兴科技股在爆发期涨幅可达10-50倍，出清期跌幅可达80-90%
+ 代表行业：互联网（1995-2000）、智能手机（2007-2015）、新能源车（2015-2025）、人工智能（2023- ）

#### 6. 供给周期（农产品/大宗商品特有）

+ 驱动因素：种植/开采决策与收获/产出之间的时滞（蛛网模型）
+ 影响范围：农产品、畜禽养殖、部分矿产
+ 典型特征：
  + 价格上涨→农户扩产→供应增加需要时间（如生猪需1.5年，橡胶需7年）
  + 供应集中释放→价格暴跌→农户减产→供应减少→价格再次上涨
  + 周期长度由供给响应时间决定
+ 股市表现：相关股票价格领先于现货价格波动，周期性强但独立于宏观
+ 代表行业：生猪养殖（猪周期约4年）、天然橡胶（约7年）、木材

#### 7. 政策/监管周期

+ 驱动因素：货币政策、财政政策、行业监管、环保标准、产业补贴
+ 影响范围：特定受政策影响大的行业
+ 典型特征：
  + 宽松周期：降准降息、减税、补贴、放松监管→行业景气上行
  + 紧缩周期：加息、加税、集采、环保限产、反垄断→行业景气下行
  + 政策周期往往与宏观周期叠加或对冲
+ 股市表现：政策敏感型行业股价在政策转向时出现剧烈波动
+ 代表行业：
  + 医药（集采政策）：集采周期下行，集采缓和/豁免则修复
  + 互联网（反垄断）：监管收紧期下跌，监管常态化后分化
  + 光伏/电动车（补贴周期）：补贴退坡期行业洗牌
  + 煤炭/钢铁（环保限产）：限产期价格上升，放开期价格回落

#### 8. 地缘/供应链周期

+ 驱动因素：贸易摩擦、技术封锁、战争、疫情、自然灾害
+ 影响范围：全球化程度高、供应链集中的行业
+ 典型特征：
  + 冲击发生：供应链中断，价格暴涨，部分企业受益（国产替代）
  + 适应调整：企业重构供应链（备库存、多元化采购、本地化生产）
  + 新均衡：成本上升，格局重塑
+ 股市表现：受影响行业出现非周期性的大涨大跌
+ 代表行业：
  + 半导体（美国对华技术封锁）：国产替代逻辑催生长周期
  + 航运（红海危机）：运价短期暴涨
  + 化肥/粮食（俄乌冲突）：价格脉冲式上涨

### 按行业周期敏感度分类的特点对比

#### 强周期行业

+ 经济相关性：正相关极强，Beta值通常1.2-1.8
+ 盈利波动：剧烈，景气高点利润是低点的数倍甚至数十倍
+ 股价特征：牛市中领涨，熊市中领跌；涨幅可达200-500%，跌幅可达50-80%
+ 估值特征：市盈率在景气低点反而很高（因利润微薄），景气高点反而很低（因利润暴增），呈现反身性
+ 投资时机：在衰退末期/复苏初期布局，繁荣后期逐步退出
+ 典型行业：有色金属、煤炭、钢铁、工程机械、航空、证券

#### 弱周期（防御性）行业

+ 经济相关性：弱正相关或接近零相关，Beta值0.5-0.8
+ 盈利波动：平稳，年度增减幅度通常在5-15%以内
+ 股价特征：熊市抗跌（跌幅通常小于大盘5-15个百分点），牛市涨幅靠后
+ 估值特征：市盈率相对稳定，熊市中估值溢价上升，牛市中估值折价
+ 投资时机：经济衰退期或市场高波动期作为避险配置
+ 典型行业：食品饮料（必需）、医药、公用事业、电信、高速公路

#### 逆周期/抗周期行业

+ 经济相关性：负相关或经济下行时表现更好
+ 盈利波动：经济下行期受益，上行期相对平淡
+ 典型逻辑：
  + 折扣零售：消费者转向低价渠道
  + 黄金：避险需求，实际利率下行
  + 二手机/维修服务：以修代买
  + 职业技术培训：失业人员再培训需求
  + 典当/小贷：资金紧张催生短期融资需求
+ 投资时机：经济衰退信号明确时配置
+ 典型行业：折扣店、黄金股、二手车拍卖、部分低端餐饮

#### 成长性行业（独立周期）

+ 经济相关性：弱相关，主要由技术渗透率、产品创新驱动
+ 盈利波动：初期高增长（30-100%年增速），中后期放缓
+ 股价特征：穿越牛熊，10年10倍常见；但在技术泡沫破裂期跌幅同样巨大
+ 估值特征：通常高市盈率（20-50倍甚至更高），基于未来预期定价
+ 投资时机：关注渗透率曲线（10-50%渗透率为黄金阶段），而非宏观经济
+ 典型行业：AI算力、新能源车（渗透率5-40%阶段）、SaaS软件、生物科技

### 三、周期叠加与识别要点

#### 多个周期同时作用的常见组合

+ *宏观经济上行 + 库存主动补库 + 产能扩张期*：工业、资源品出现“三击”行情，涨幅巨大
+ *宏观经济衰退 + 主动去库存 + 产能过剩*：强周期行业出现“三杀”，深跌
+ *宏观经济衰退 + 技术创新爆发*：新兴成长行业可能独立走牛（如2008年iPhone发布后苹果逆势上涨）
+ *房地产长周期下行 + 宏观短周期上行*：房地产链相关行业反弹乏力，与大盘走势背离

#### 判断当前所处周期的实用指标

+ 宏观周期：PMI、社融增速、利率曲线、失业率
+ 库存周期：产成品库存增速、PPI、企业库销比
+ 产能周期：固定资产投资增速、产能利用率、设备工器具购置增速
+ 房地产周期：新开工面积、销售面积、居民中长期贷款、房价同比
+ 技术周期：渗透率（新产品占潜在市场比例）、技术成熟度曲线（Gartner曲线）



竞争格局
====

波特五力模型
----

+ *潜在进入者的威胁*
  + 看行业壁垒：是否存在规模经济、专利技术、牌照管制、品牌忠诚度等。
  + 若壁垒低，新企业容易进入，会压制行业利润率。
+ *供应商的议价能力*
  + 上游集中度高、产品差异化大、转换成本高 → 供应商话语权强，侵蚀行业利润。
  + 反之，行业对上游有较强的成本控制能力。
+ *购买者的议价能力*
  + 下游客户集中、产品同质化、转换成本低 → 客户压价能力强，行业盈利承压。
  + 品牌消费品、高转换成本的B2B软件等，客户议价能力相对弱。
+ *替代品的威胁*
  + 替代品性能价格比越好、转换成本越低，威胁越大。
  + 例如：新能源对传统燃油的替代；视频会议对部分商务出差的替代。
+ *行业内现有竞争者的竞争强度*
  + 竞争者数量多且势均力敌 → 价格战、营销战频繁。
  + 行业增速慢 → 争夺存量市场，竞争加剧。
  + 产品同质化、退出壁垒高 → 竞争更加激烈。


定量判断指标
----

+ 行业集中度（CRn，如CR3/CR5/CR10）
  + CRn 高（如 CR3 > 70%）→ 寡占或垄断格局，领先企业利润率高且稳定。
  + CRn 低 → 竞争型格局，企业盈利波动大。
+ 赫芬达尔指数（HHI）
  + HHI > 2500 → 高度集中市场（适合大龙头）。
  + HHI < 1500 → 竞争市场，需选差异化标的。
+ 前几名企业市场份额的变化趋势
  + 份额持续向头部集中 → 竞争格局优化，龙头受益。
  + 头部份额下降或混乱 → 格局恶化或混战阶段。
+ 行业利润率水平及波动
  + 平均净资产收益率（ROE）、毛利率长期较高且稳定 → 格局良好。
  + 毛利率剧烈下滑 → 价格战或供需恶化信号。


典型竞争格局
----

+ 完全垄断（一家独大）
  + 特征：政府特许或自然垄断（水电、部分公用事业）。
  + 投资逻辑：稳定但增长空间有限，看股息和价格管控。
+ 寡头垄断（少数几家主导）
  + 特征：CR3 > 60%~70%，且有默契或非价格竞争（如家电、白电、部分化工）。
  + 投资逻辑：行业盈利稳定，龙头企业优先，关注份额变化和提价能力。
+ 垄断竞争（品牌/差异化的众多企业）
  + 特征：集中度较低，产品有差异化（如服装、餐饮、化妆品）。
  + 投资逻辑：关键在于品牌、渠道、产品创新；易出黑马，但波动大。
+ 完全竞争（价格战红海）
  + 特征：高度分散、同质化、低门槛（如普通农产品、低端加工）。
  + 投资逻辑：谨慎回避，除非出现颠覆性整合者或极端成本优势企业。


动态竞争格局
----

+ 行业生命周期阶段与格局演变
  + 导入期：格局混乱，关注技术路线胜出者。
  + 成长期：高增速掩盖竞争，格局未定型；需看谁扩产快、份额提升。
  + 成熟期：格局固化，龙头优势凸显；关注价格战是否结束。
  + 衰退期：格局出现“二次集中”，存活企业可能受益于退出。
+ 产能与供需周期
  + 产能过度扩张 → 供过于求 → 竞争恶化 → 盈利低点。
  + 产能出清/退出 → 供需改善 → 格局转折 → 盈利拐点。
+ 技术或商业模式变革的影响
  + 变革可能颠覆原有格局（如新能源车对传统车）。
  + 也可能巩固龙头地位（如规模效应+智能化）。


构建简单的分析清单
----

+ 行业目前CR3/CR5是多少？过去3年趋势？
+ 是否有明显的价格战或非理性竞争行为？
+ 头部企业与尾部企业的ROE差距是否在拉大？
+ 客户/供应商的集中度及议价能力？
+ 是否存在较强替代品？替代速度如何？
+ 新进入者面临的主要壁垒是什么（资金、技术、牌照、渠道）？
+ 管理层是否理性竞争（不盲目扩产/打价格战）？
+ 近一轮行业低谷时，是否有企业破产或被并购？


操作上的建议
----

+ 历史对比：该行业当前格局比三年前更集中还是更分散？
+ 国际对比：该行业在中国 vs 成熟市场的集中度差异，是否可能走向类似结构？
+ 上下游对比：格局比上游好还是比下游差？决定利润在链条中的分配。



政策影响
====

识别核心目标与工具
----

+ 判断政策类型：是财政政策（如补贴、减税）、货币政策（如利率、准备金率）、产业政策（如“十四五”规划），还是监管政策（如反垄断、数据安全）。
+ 提取关键词：如“碳中和”、“自主可控”、“集采”、“促消费”等，这些词汇直接指向相关行业。
+ 明确政策力度：看是鼓励性（鼓励发展）、约束性（限产、提标），还是中性规范（制定标准）。


定位直接作用的行业
----

+ 找出政策中明确提及的行业：如新能源政策直接利好光伏、风电、电动车产业链。
+ 找出被政策直接限制的行业：如“房住不炒”直接影响房地产开发、建材。
+ 分析受益逻辑：补贴增加利润、税收优惠提升净利率、准入限制减少竞争等。
+ 分析受损逻辑：成本上升、需求下降、合规成本增加。


分析产业链传导
----

+ 上游传导：政策刺激下游需求（如电动车），则会拉动上游锂、钴、镍等资源，以及中游电池、电控。
+ 下游传导：政策压制上游（如限产钢铁），则会推高下游用钢行业（如工程机械、汽车）的成本。
+ 替代与互补效应：鼓励A行业，会压制其替代品B行业，并带动其互补品C行业（如充电桩是电动车的互补品）。
+ 基础设施与配套：重大产业政策往往拉动电网、数据中心、物流等基础服务行业。


行业供需与竞争格局
----

+ 判断政策是否改变行业供需：补贴可能扩大需求，限产会收缩供给，从而影响价格与利润。
+ 评估政策对竞争格局的影响：严格的环保政策可能淘汰小企业，利好龙头集中度提升；普惠性补贴可能吸引大量新进入者，加剧竞争。
+ 分析政策执行的节奏与力度：是立即生效还是逐步推进？是否存在地方配套差异？这影响行业受益的时间窗口。


区分短期情绪冲击与长期结构变化
----

+ 短期影响：消息公布后，市场情绪驱动的股价波动，可能过度反应。需关注政策是否改变未来3-6个月的基本面。
+ 长期影响：是否改变行业的技术路径、商业模式或成本曲线。例如，医保集采对仿制药是长期利润压制，但对创新药是长期鼓励。
+ 预期差分析：如果政策已被市场充分预期（如已体现在前期涨幅中），则落地后可能“利好出尽”；超预期部分才是关键。


交叉验证与排除干扰
----

+ 对比政策前后的行业数据：如产量、库存、价格、企业订单，验证逻辑是否成立。
+ 关注国际环境与配套政策：例如，国内鼓励芯片，但美国出口管制可能削弱效果；同一行业中，不同子行业可能受益程度相反（如传统能源 vs 新能源）。
+ 警惕“口号型政策”：缺乏细则、预算或考核机制的政策影响有限。


常见政策影响行业
----

+ 宽松货币政策 → 利好金融（银行息差变化需具体分析）、高负债行业（地产、基建）、成长股估值（科技、医药）。
+ 产业扶持政策（如半导体大基金） → 直接利好半导体设备、材料、设计；间接利好下游电子产品；但需注意估值透支风险。
+ 环保限产政策 → 直接利好行业龙头（如钢铁、水泥、化工），因小厂关停；直接利空下游建筑、制造企业成本端。
+ 药品/耗材集采政策 → 利空仿制药、低值耗材企业；利空非中选企业；相对利好创新药、CXO（医药合同研发生产服务）以及药店渠道。
+ 平台经济反垄断 → 利空互联网巨头（尤其是依靠垄断定价的广告、佣金业务）；利好中小商户、其他消费渠道；长期利好数据安全与合规服务商。


实操建议
----

+ 建立政策事件跟踪表，记录政策名称、发布时间、核心条款、利好/利空行业及逻辑。
+ 将政策影响与公司财务模型结合：例如，判断减税政策能提升目标公司净利润的具体百分比。
+ 利用行业指数（如申万、中信行业分类）与政策事件进行历史复盘，观察板块相对收益的走势。



产业链分析
====

在证券投资分析中，分析产业链对行业的影响，核心是理解*价值流动、议价能力、利润分布*以及*外部冲击的传导机制*。

明确产业链结构
----

+ 上游：原材料、能源、零部件、基础技术（如锂矿、芯片设计、原油）
+ 中游：制造、加工、组装、系统集成（如电池制造、晶圆代工、汽车零部件）
+ 下游：品牌、分销、渠道、终端服务（如整车厂、零售药店、云计算服务商）
+ 辅助环节：物流、设备供应商、技术服务商、金融支持


分析核心维度
----

### 利润分布与“微笑曲线”

+ 高利润区：通常位于上游的稀缺资源/核心技术，以及下游的品牌/渠道。中游制造易陷入红海竞争。
+ 低利润区：同质化、低壁垒的加工组装环节。
+ 投资含义：优先关注拥有高壁垒的上游资源或技术，以及具有品牌溢价或客户粘性的下游企业。

### 议价能力与集中度

+ 供应商议价能力：若上游高度集中（如少数矿企），中下游利润易被挤压。
+ 购买商议价能力：若下游渠道集中（如大型商超、电商平台），中上游需让利。
+ 行业内竞争：中游若过于分散，价格战频发；若形成寡头，可向上下游传导压力。
+ 投资含义：寻找产业链中集中度提升或已形成寡头的环节。

### 依赖关系与瓶颈环节

+ 关键瓶颈：某项技术、材料或零部件若无可替代，其供应商有极强定价权（如光刻机、高端轴承钢）。
+ 替代风险：若下游可轻易更换供应商，则该环节护城河低。
+ 投资含义：识别并投资于产业链的“咽喉”位置。

### 价值传递与成本传导

+ 成本转嫁能力：通胀时，议价强的环节可将成本上涨转嫁给上下游；弱环节则自我消化。
+ 价格弹性：下游需求若缺乏弹性，中上游提价更容易传导。
+ 投资含义：在成本上升周期，选择能提价且销量不受影响的企业。


关键指标量化
----

+ 毛利率差异：比较上中下游的毛利率，识别价值高地。
+ 应收账款/应付账款周转：反映企业对上下游的资金占用能力（占款能力强 → 产业链地位高）。
+ 库存水平：中游库存积压可能预示下游需求走弱。
+ 产能利用率：瓶颈环节常维持高利用率。


分析动态变化机制
----

### 技术变革

+ 颠覆上游（如合成生物替代天然提取）
+ 消灭中游（如3D打印减少零件加工）
+ 重构下游（如电商冲击实体店）
+ 投资行动：判断新技术的受益/受损环节。

### 政策与外部冲击

+ 关税/贸易壁垒：影响跨国产业链布局，改变利润在国别间的分配。
+ 环保政策：提高上游成本，加速中游出清。
+ 产业补贴：可能人为催生中游过剩产能，压低利润。
+ 自然灾害/地缘政治：暴露供应链脆弱环节，短期利好库存充足者，长期促进国产替代。

### 产业链迁移与重构

+ 工厂从高成本区域迁至低成本区域（影响就业与税收，改变上市公司成本结构）。
+ 近岸外包、友岸外包（影响物流成本与供应稳定性）。
+ 投资含义：关注迁移中的受益者（如越南建厂企业）与受损者（如原产地出口商）。


分析具体步骤
----

1. 画出简化产业链图谱（上中下游各列出2-3个子环节）
2. 计算各环节头部公司的平均毛利率、净利率（定位利润中心）
3. 访谈或调研：询问“如果缺货/涨价，谁最着急？”
4. 历史复盘：回顾过去一次原材料大涨/需求崩溃时，各环节股价先后顺序和幅度
5. 压力测试：假设芯片短缺/运费翻倍/汇率波动，推算各环节利润变动
6. 跟踪关键数据：
   + 上游：大宗商品价格、产量、库存
   + 中游：开工率、订单量、成品库存
   + 下游：终端销量、渠道库存、客单价


常见产业链形态及其投资启示
----

+ 金字塔型（上游寡头+下游分散）：投资上游资源龙头
+ 哑铃型（上、下游集中，中游松散）：避开中游，投资两端
+ 平铺型（各环节皆分散）：寻找规模化整合者
+ 链主驱动型（如苹果、特斯拉）：投资链主本身，或链主扶持的核心供应商
+ 本地化短链（如餐饮、水泥）：重点看区域市场份额和运输半径


注意事项
----

+ 不要静态看待产业链：企业的垂直整合或外包会改变自身在链条中的位置。
+ 警惕“伪龙头”：某些环节看似集中，实则产品无差异（如大宗化学品）。
+ 区分短期扰动与长期结构变化：运费暴涨通常不会改变长期利润分配格局。
+ 全球产业链需叠加地缘政治风险：不能只看效率，还要看安全冗余。
