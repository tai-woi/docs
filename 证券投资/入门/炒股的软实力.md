基础认知力
====

这个层级的目标是*不被忽悠*，能分辨什么是常识，什么是鬼故事。

+ 失败时的表现：*被消息牵着走*
+ 市场规则：交易规则（T+1、涨跌停）、手续费结构、打新规则等。
+ 公司分析：看懂财报（三张表）、估值指标（PE、PB、ROE）、行业景气度判断。
+ 技术工具：理解K线语言、量价关系、2-3个常用指标（如均线、MACD）的原理，而不是背口诀。
+ 宏观常识：利率、CPI、汇率是如何传导到股市的。
+ 成功时的表现：*能看懂基本面*


策略构建力
====

这个层级的目标是*可复制*——赚要赚得明明白白，亏要亏在规则之内。

+ 失败时的表现：*随意买卖，凭感觉*
+ 选股逻辑：你是做价值投资（看低估）、成长投资（看增速），还是趋势跟踪（看动量）？必须选一条主航道。
+ 买卖规则：不是“觉得差不多了就买”，而是设定具体的触发条件（比如站上60日均线买，跌破止损线卖）。
+ 仓位管理：永远别一把梭。懂得金字塔建仓、倒金字塔减仓，知道什么时候该重仓，什么时候该轻仓。
+ 成功时的表现：*有明确的交易系统*


情绪驾驭力
====

这是区分“业余”和“专业”的分水岭。这个层级的目标是*知行合一*——你的操作不受当日盈亏的情绪干扰。

+ 失败时的表现：*追涨杀跌，后悔自责*
+ 对抗贪婪：大涨时管住手，不因“怕错过”而追高。
+ 对抗恐惧：大跌时保持理智，不因“怕亏损”而割在地板。
+ 对抗懊悔：卖飞了不拍大腿，买早了不焦虑，接受不完美。
+ 对抗希望：不因被套就“装死”并幻想奇迹，客观面对事实。
+ 成功时的表现：*操作平稳，心如止水*



动态决策力
====

市场是活的，你的脑子也得是活的。这个层级的目标是*随势而动*，像水一样，根据地形（市场环境）改变形状。

+ 失败时的表现：*死扛或死空，不懂变通*
+ 灰度思维：接受股市没有“一定”，只有“概率”。能持有“可能错”的仓位安然入睡。
+ 快速纠偏：当买入逻辑被证伪（比如业绩暴雷、政策转向），能立刻认错离场，不纠结成本。
+ 噪音过滤：面对铺天盖地的小作文和股评，能迅速判断哪些是有效信息，哪些是情绪垃圾。
+ 周期感知：能大致感知市场处于什么阶段（冰点、回暖、高潮、退潮），而不是只盯着自己手里的股。
+ 成功时的表现：*顺势而为，灵活应对*


认知进化力
====

这是让你在股市活过10年、20年的根本保障。这个层级的目标是*持续迭代*——把股市当作修炼场，用一辈子的时间慢慢优化自己。

+ 失败时的表现：*年年犯同样的错*
+ 自省习惯：每笔交易后写复盘笔记，不找借口，只找漏洞。赚了要问“是运气还是实力”，亏了要问“是概率事件还是逻辑错误”。
+ 开放心态：不固守一种风格。当市场生态巨变（比如注册制、量化主导）时，愿意抛弃旧经验，学习新范式。
+ 生活分离：股市只是生活的一部分，而不是全部。能用闲钱投资，不借钱、不加杠杆，睡得着觉。
+ 成功时的表现：*犯过的错不再犯第二次*
