股票的本质
====

+ 股票是公司所有权的凭证（股东权益）
+ 股价由“内在价值”与“市场情绪”共同驱动
+ 交易是“所有权转移”，零和博弈（不计分红与费用）


核心交易规则
====

+ 交易时间（盘前/盘中/盘后）
+ 订单类型（市价单、限价单、止损单、止损限价单）
+ T+0 / T+1 结算制度（不同市场差异）
+ 涨跌停板机制（若适用）
+ 交易费用构成（佣金、印花税、过户费、滑点）


关键市场参与者
====

+ 散户（你）
+ 机构（基金、对冲基金、养老金）
+ 做市商与高频交易者
+ 公司内部人（增减持）
+ 监管机构（SEC/证监会）


基础分析工具与术语
====

+ K线图（开盘/收盘/最高/最低）
+ 成交量与成交额
+ 均线（MA5、MA20、MA60）
+ 常见技术指标（RSI、MACD、布林带）——先理解含义，不急于使用
+ 盘口数据（买卖五档、内外盘）


基本面和技术面
====

+ 基本面：市盈率、市净率、ROE、营收增长、现金流、行业前景
+ 技术面：趋势、支撑/阻力、形态、量价关系
+ 明确两者作用不同：基本面选标的，技术面找时机


风险管理优先于盈利
====

+ 单笔交易最大亏损比例（如1%~2%总资金）
+ 止损位的设定逻辑（非随意数字）
+ 仓位管理（金字塔/倒金字塔/等额）
+ 相关性分散（不同行业/资产类别）
+ 避免“加仓摊平亏损”的陷阱


交易心理与行为偏误
====

+ 恐惧与贪婪循环
+ 确认偏误（只找支持自己观点的信息）
+ 处置效应（过早止盈、过晚止损）
+ 从众心理（FOMO）
+ 建立交易纪律手册（写在纸上）


个人交易框架
====

+ 交易周期（日内/波段/趋势/价值投资）
+ 选股逻辑（市值范围、流动性要求）
+ 入场与出场的前提条件（必须量化，非“感觉”）
+ 交易记录模板（日期、标的、方向、价格、理由、盈亏、情绪状态）


从模拟盘开始
====

+ 模拟盘目的：熟悉操作、测试纪律，而非追求收益
+ 至少完成20~50笔模拟交易
+ 复盘每一笔，统计胜率、盈亏比、最大回撤


持续学习的知识树
====

+ 宏观（利率、通胀、GDP）
+ 行业逻辑（周期/非周期、成长/防御）
+ 财报解读能力（三张表）
+ 事件驱动（财报季、政策发布、地缘政治）
+ 逐步添加“经济日历”跟踪习惯


交易与投资的区别
====

+ 投资：基于价值，长期持有，淡化波动
+ 交易：基于价格波动，依赖纪律与概率
+ 初期建议先选一条路，避免混合混乱


现实预期与退出机制
====

+ 年化收益目标（如跑赢通胀+5%，而非翻倍）
+ 最大可接受回撤（总账户）
+ 何时停止交易（连续亏损、情绪失控、重大生活事件）
+ 定期评估（月度/季度）是否适合继续交易
